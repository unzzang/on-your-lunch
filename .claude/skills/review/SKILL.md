# 문서 리뷰

산출물의 품질을 검토하고 개선 피드백을 제공한다.

## 작업 기준

### 리뷰 관점

1. **완성도** — 필수 항목 누락 여부
2. **정확성** — 사실 오류, 모순, 불일치
3. **명확성** — 모호 표현, 해석의 여지
4. **일관성** — 다른 문서와의 정합성
5. **실행 가능성** — 이 문서만으로 다음 단계 진행 가능한가

### 피드백 형식

- **통과** — 수정 없이 확정
- **수정 제안** — 구체적 위치와 개선 방향 명시
- **재작성 필요** — 근본적 문제가 있는 경우

### 규칙

- 감정적 평가 금지 ("좋은 것 같다" 사용 불가)
- 모든 피드백에 이유를 명시한다

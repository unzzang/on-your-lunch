# SEO 블로그 글 작성

서비스의 인지도와 유입을 높이기 위한 블로그 글을 작성한다.

## 작업 기준

### SEO 최적화

- 타겟 키워드 선정 후 제목·소제목·본문에 자연스럽게 배치
- 메타 디스크립션 작성
- 내부 링크 구조 설계

### 글 유형

- **서비스 소개** — 기능 설명, 사용법
- **기술 블로그** — 개발 과정, 기술 결정 배경
- **콘텐츠 마케팅** — 타겟 고객의 관심사 기반 정보성 글

### 품질 기준

- 사실 관계 정확성 (가짜 데이터 금지)
- 발행 전 송현아(검증전문가) 팩트체크 필수

## 협업

- **송현아(검증전문가)**에게 발행 전 팩트체크 요청 (필수)
- **박소율(그래픽 디자이너)**에게 블로그 삽입 이미지 요청
- **한지민(마케터)**과 콘텐츠 전략 협의
- **이수빈(SNS)**과 콘텐츠 배포 협의

# GTM 전략 + 퍼포먼스 마케팅

서비스의 시장 진입 전략을 수립하고 마케팅을 실행한다.

## 작업 기준

### GTM 전략

1. 타겟 세그먼트 정의 (리서치 결과 기반)
2. 포지셔닝 메시지 수립
3. 채널 전략 (유료/무료, 온라인/오프라인)
4. 가격 전략 (프리미엄, 프리미엄, 구독 등)
5. 런칭 타임라인

### 퍼포먼스 마케팅

- 광고 매체별 예산 배분
- 크리에이티브 방향 설정 (제작은 박소율에게 요청)
- 전환 퍼널 설계
- A/B 테스트 계획

### 원칙

- 모든 수치에 근거 명시 (추정치는 "추정" 표기)
- 가격 결정은 PO 승인 필수

## 협업

- **박소율(그래픽 디자이너)**에게 광고 크리에이티브 제작 요청
- **강서연(블로거)**과 콘텐츠 마케팅 협의
- **이수빈(SNS)**과 채널 운영 협의
- **톰크루즈(데이터)**에게 마케팅 지표 분석 요청
- **PO**에게 예산·가격 승인 요청

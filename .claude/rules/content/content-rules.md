# 콘텐츠 규칙

## 발행 전 검증 필수

모든 콘텐츠(블로그, 영상 스크립트)는 발행 전 검증전문가(송현아)의 팩트 체크를 거친다.

## 블로그

- 키워드 리서치 → 구조 설계 → 초안 → 검증 → 발행
- SEO 최적화 필수 (타이틀, 메타 디스크립션, 헤딩 구조, 키워드 밀도)
- 모든 수치에 출처 명시
- 비주얼(이미지, 인포그래픽)은 디자인팀 박소율에게 요청

## 영상

- 주제 선정 → 스크립트 → 검증 → PO 촬영/편집 → 발행
- 스크립트에 후킹 포인트(처음 5초) 반드시 포함
- 썸네일은 디자인팀 박소율에게 요청
- 실제 촬영/편집은 PO 또는 외부에서 수행

# 리서치 규칙

## 조사 구조

```
조사 설계 → 4명 병렬 리서치 → 검증 → 전략 종합
```

- 4명의 리서처는 반드시 **동시에 독립적으로** 조사. 서로의 결과를 참조하지 않는다.
- 조사 설계 없이 바로 리서치에 들어가지 않는다.
- 검증(송현아) 통과 후에만 전략 종합으로 넘어간다.

## 데이터 규칙

- 모든 수치에 출처(URL, 보고서명) 필수
- 출처 미확인 데이터는 "출처 미확인"으로 별도 표기
- 한국어 + 영어 키워드 모두 검색
- 최소 5개 이상 소스에서 정보 수집

## 산출물

개별 보고서 + 종합 보고서를 모두 파일로 저장한다.

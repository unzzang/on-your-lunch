## 1. 스플래시 / 인트로

### 스플래시
- [ ] 앱 실행 → 온유어런치 로고 + "매일 점심, 고민 끝." 태그라인 표시
- [ ] 스플래시 1.5초 후 자동 전환
- [ ] 로그인 유지 상태 → 스플래시 후 홈 화면 직행
- [ ] 비로그인 상태 → 스플래시 후 인트로/로그인 화면 표시

### 인트로/로그인
- [ ] 인트로 일러스트 + "매일 점심 뭐 먹지? 온유어런치가 골라줄게요!" 텍스트 표시
- [ ] "Google로 시작하기" 버튼 표시 (Google 브랜드 가이드라인 준수)
- [ ] 하단 "계속하면 이용약관에 동의하게 됩니다" 텍스트 + 약관 링크 표시

## 2. 로그인 / 인증

### Google 소셜 로그인
- [ ] Google 로그인 버튼 클릭 → Google OAuth 인증 → 신규 사용자 → 약관 동의 화면 이동
- [ ] Google 로그인 버튼 클릭 → Google OAuth 인증 → 기존 사용자 → 홈 화면 직행
- [ ] Google 로그인 시 이메일, 닉네임(Google 표시 이름), 프로필 사진 수집 확인
- [ ] dev-login 버튼 클릭 → 즉시 홈 이동 (개발 환경에서만 노출)

### 로그인 에러
- [ ] Google OAuth 인증 오류 → "로그인에 실패했어요. 다시 시도해주세요." 토스트 + 재시도 버튼
- [ ] Google 인증 창에서 사용자 취소 → 로그인 화면으로 복귀 (별도 메시지 없음)
- [ ] 네트워크 오류 → "인터넷 연결을 확인해주세요." + 재시도 버튼

### 토큰 관리
- [ ] Access Token 1시간 만료 → 자동 갱신 (Refresh Token 사용)
- [ ] Refresh Token 30일 만료 → "다시 로그인해주세요." → 로그인 화면 이동
- [ ] 로그아웃 시 클라이언트 토큰 삭제 + 서버 Refresh Token 무효화

## 3. 약관 동의

- [ ] "전체 동의" 체크 → 모든 항목(필수 3개 + 선택 1개) 선택됨
- [ ] "전체 동의" 해제 → 모든 항목 해제됨
- [ ] 필수 3개(서비스 이용약관, 개인정보 처리방침, 위치정보 이용약관) 모두 체크 → "동의하고 시작하기" 버튼 활성화
- [ ] 필수 항목 1개라도 미체크 → "동의하고 시작하기" 버튼 비활성화
- [ ] 마케팅 수신 동의(선택) 미체크 상태에서도 필수 3개 체크 시 → 버튼 활성화
- [ ] 각 약관 항목 [>] 탭 → 약관 전문 웹뷰로 이동
- [ ] "동의하고 시작하기" 탭 → 온보딩 Step 1로 이동

## 4. 온보딩

### 공통
- [ ] 상단 프로그레스 바 표시 (Step 1/3, 2/3, 3/3)
- [ ] 각 Step에서 "이전"(뒤로) 버튼 → 이전 Step으로 이동
- [ ] Step 3만 "건너뛰기" 링크 표시

### Step 1: 회사 위치 설정
- [ ] "회사가 어디인가요?" 타이틀 + 설명 텍스트 표시
- [ ] 검색 바 탭 → 키보드 올라오고 자동완성 결과 표시
- [ ] 건물명 또는 도로명 주소 검색 → 검색 결과 리스트 표시
- [ ] 검색 결과 선택 → 지도에서 해당 위치로 이동 + 핀 표시
- [ ] 지도 직접 탭 → 핀 위치 변경 가능
- [ ] 하단에 선택된 주소 표시 (도로명 주소 + 건물명)
- [ ] 주소 미선택 상태 → "다음" 버튼 비활성화
- [ ] 주소 선택 완료 → "다음" 버튼 활성화 → 탭 시 Step 2로 이동

### Step 1 에러
- [ ] 위치 검색 결과 0건 → "검색 결과가 없어요. 다른 키워드로 검색해보세요."
- [ ] 지도 핀 찍기 시 위치 권한 미허용 → "현재 위치를 찾으려면 위치 권한이 필요해요." + 설정 이동 버튼
- [ ] 위치 권한 거부 시에도 주소 검색으로 대체 가능

### Step 2: 취향 + 가격대 설정
- [ ] "어떤 음식을 좋아하세요?" 타이틀 표시
- [ ] 7개 카테고리 칩 표시: 한식, 중식, 일식, 양식, 아시안, 분식/간편식, 샐러드/건강식
- [ ] 카테고리 칩 탭 → 색상 채워짐(활성화), 재탭 → 해제. 복수 선택 가능.
- [ ] "점심 예산은?" 가격대 3개 탭 표시: 1만원 이하 / 1~2만원 / 2만원 이상
- [ ] 가격대 단일 선택 (기본값 없음)
- [ ] 카테고리 0개 선택 + "다음" 탭 → "좋아하는 음식을 1개 이상 선택해주세요!" 토스트
- [ ] 가격대 미선택 상태 → "다음" 버튼 비활성화
- [ ] 카테고리 1개 이상 + 가격대 1개 선택 → "다음" 버튼 활성화 → 탭 시 Step 3로 이동

### Step 3: 제외 설정
- [ ] "빼고 싶은 음식이 있나요?" 타이틀 + "없으면 건너뛰어도 돼요!" 설명 표시
- [ ] 알레르기 6개 칩 표시: 갑각류, 견과류, 유제품, 밀, 달걀, 대두 (복수 선택)
- [ ] 싫어하는 카테고리 7개 칩 표시 (복수 선택)
- [ ] 아무것도 선택 안 해도 "완료" 버튼 활성 상태
- [ ] "건너뛰기" 링크 탭 → 아무 선택 없이 홈 화면 이동
- [ ] "완료" 버튼 탭 → 선택 항목 저장 후 홈 화면 이동
- [ ] 온보딩 완료 후 홈 화면에 오늘의 추천 3장 표시

### 온보딩 중 네트워크 오류
- [ ] 인터넷 끊김 → "인터넷 연결을 확인해주세요." + 재시도 버튼

## 5. 홈 (점심 추천)

### 상단 바
- [ ] 온유어런치 로고 탭 → 홈 최상단으로 스크롤
- [ ] 알림 아이콘 표시 (MVP에서는 비활성)

### 인사 영역
- [ ] "{닉네임}님, 오늘 점심 뭐 먹을까요?" 텍스트 표시
- [ ] 스크롤 시 상단 바에 가려짐

### 필터 영역
- [ ] 카테고리 칩 가로 스크롤 표시 (한식/중식/일식/양식/아시안/분식/간편식/샐러드/건강식)
- [ ] 카테고리 칩 탭 → 활성화(색상 채워짐), 재탭 → 해제
- [ ] 카테고리 복수 선택 가능
- [ ] 도보 거리 탭 3개 표시: 5분 / 10분(기본값) / 15분. 단일 선택.
- [ ] 가격대 탭 3개 표시: 1만원 이하 / 1~2만원 / 2만원 이상. 단일 선택. 기본값 없음(전체).
- [ ] 필터 변경 → 추천 카드 즉시 갱신 (새로고침 횟수 차감 안 함)
- [ ] 필터 영역 스크롤 시 상단 고정(sticky)
- [ ] 필터 초기화 → 기본 추천으로 복귀

### 추천 카드
- [ ] 세로로 3장 나열 표시 (스크롤)
- [ ] 각 카드에 표시: 대표 사진, 식당명(볼드), 카테고리, 도보 시간, 가격대
- [ ] 즐겨찾기 하트 아이콘 카드 우상단 표시 (♥ 추가됨 / ♡ 미추가)
- [ ] 즐겨찾기 하트 탭 → 토글 (화면 유지)
- [ ] 이전 방문 식당 → "★ 4.0 · 3번 방문" 뱃지 표시
- [ ] 대표 사진 없는 식당 → 카테고리별 기본 일러스트 표시
- [ ] 카드 탭 → 식당 상세 화면 이동

### 추천 새로고침
- [ ] 추천 카드 3장 아래 "다른 추천 보기 (남은 횟수/5)" 버튼 표시
- [ ] 새로고침 탭 → 새로운 3장 표시 (카드 페이드 아웃 → 새 카드 페이드 인)
- [ ] 새로고침 시 이전 추천 식당과 중복 없음
- [ ] 남은 횟수 표시 정확 (예: "다른 추천 보기 (4/5)")
- [ ] 5회 소진 → 버튼 비활성화 + "오늘 추천을 모두 사용했어요. 탐색에서 직접 찾아보세요!" + 탐색 탭 이동 링크
- [ ] 하루 기준 매일 자정(00:00) 초기화

### 하단 탭 바
- [ ] 4개 탭 표시: 홈(집), 탐색(돋보기), 이력(캘린더), 마이(사람)
- [ ] 현재 활성 탭 아이콘 + 텍스트 색상 강조
- [ ] 화면 하단 고정
- [ ] 각 탭 탭 → 해당 화면 이동

### 홈 상태별 화면
- [ ] 정상 상태: 추천 카드 3장 + 필터 + 새로고침 버튼
- [ ] 추천 후보 0개 → "조건에 맞는 식당을 찾지 못했어요" + 필터 초기화 버튼
- [ ] 새로고침 소진 → 새로고침 버튼 비활성 + 탐색 안내
- [ ] 네트워크 오류 → "인터넷 연결을 확인해주세요" + 재시도 버튼
- [ ] 첫 사용 (이력 0건) → 정상 표시 (이력 없이도 취향 기반 추천)

### 추천 로직 (서버)
- [ ] 회사 위치 기준 도보 10분(기본값) 이내 식당 필터링
- [ ] 사용자 취향 설정(좋아하는 카테고리, 알레르기, 가격대) 반영
- [ ] 최근 5일간 먹은 식당 제외
- [ ] 최근 3일간 먹은 카테고리 가중치 하향
- [ ] 조건 완화 단계별 진행 (도보 거리 확대 → 이력 제외 기간 단축 → 카테고리 가중치 제거 → 이력 제외 해제)
- [ ] 4단계까지 완화 후에도 0개 → 에러 메시지 표시

### 위치 미설정 에러
- [ ] 회사 위치 미설정 상태에서 추천 요청 → "먼저 회사 위치를 설정해주세요!" + 위치 설정 화면 이동 버튼

## 6. 식당 상세

### 상단 바
- [ ] 뒤로 버튼(좌측) 탭 → 이전 화면(홈/탐색)으로 복귀
- [ ] 공유 버튼(우측) 탭 → 카카오톡 공유 실행
- [ ] 상단 바가 사진 영역 위에 반투명 오버레이로 표시

### 사진 영역
- [ ] 풀 너비, 높이 화면의 약 1/3 표시
- [ ] 사진 여러 장 → 좌우 스와이프 + 하단 인디케이터 표시
- [ ] 사진 없음 → 카테고리별 기본 일러스트 1장 표시
- [ ] 위로 스크롤 시 사진 영역 축소 (패럴랙스 효과)

### 기본 정보 영역
- [ ] 식당명 (볼드, 큰 텍스트) 표시
- [ ] 카테고리 표시 (예: "한식 · 백반")
- [ ] 즐겨찾기 ♥ 아이콘 (식당명 우측) → 탭 시 토글
- [ ] 도보 시간 표시 (예: "도보 8분")
- [ ] 주소 (도로명) 표시 → 탭 시 클립보드 복사 + "주소가 복사되었어요" 토스트
- [ ] 가격대 표시 (예: "8,000~12,000원") / 가격 정보 없으면 "가격 정보 준비 중"
- [ ] 영업시간 표시 (데이터 있을 때만) / 없으면 항목 자체 미표시
- [ ] 전화번호 표시 (데이터 있을 때만) → 탭 시 전화 앱 연결 / 없으면 항목 자체 미표시

### 내 기록 영역
- [ ] 방문한 적 있는 식당 → 평균 별점, 총 방문 횟수, 최근 방문 날짜 + 메모 표시
- [ ] 방문한 적 없는 식당 → "내 기록" 영역 미표시
- [ ] 내 기록 영역 탭 → 이 식당의 전체 방문 기록 바텀시트 표시

### 약도 (지도)
- [ ] 카카오맵 SDK 정적 지도 표시
- [ ] 식당 위치 핀 + 내 회사 위치 핀 동시 표시
- [ ] 두 지점이 모두 보이는 줌 레벨 자동 조정
- [ ] 지도 탭 → 카카오맵 앱으로 이동

### 하단 고정 버튼
- [ ] "길찾기" + "먹었어요" 2개 버튼 가로 균등 배치, 하단 고정
- [ ] "길찾기" 탭 → 외부 지도 앱 실행 (카카오맵 우선 → 네이버 지도 → 웹 브라우저)
- [ ] 출발지: 사용자 회사 위치, 도착지: 식당 주소
- [ ] "먹었어요" 탭 → 먹었어요 바텀시트 표시
- [ ] 당일 이미 기록한 식당 → "먹었어요" 버튼 → "기록 수정" 버튼으로 변경

### 길찾기 에러
- [ ] 카카오맵 앱 설치됨 → 카카오맵 앱으로 이동
- [ ] 카카오맵 미설치 + 네이버 지도 설치됨 → 네이버 지도로 이동
- [ ] 둘 다 미설치 → 웹 브라우저로 카카오맵 열기

### 식당 상세 상태별 화면
- [ ] 전체 데이터 있음 → 모든 영역 정상 표시
- [ ] 사진 없음 → 기본 일러스트
- [ ] 메뉴/가격 없음 → "가격 정보 준비 중"
- [ ] 영업시간 없음 → 해당 항목 미표시
- [ ] 방문 이력 없음 → "내 기록" 영역 미표시
- [ ] 당일 이미 기록 → "기록 수정" 버튼으로 변경

## 7. 먹었어요 기록

### 먹었어요 바텀시트 (식당 상세에서 진입)
- [ ] 드래그 핸들 표시
- [ ] "{식당명}에서 먹었군요!" 타이틀 표시
- [ ] 별점 입력 영역: 1~5점, 별 아이콘 탭 (정수만, 0.5점 단위 없음)

## 22. 회원 탈퇴

- [ ] 마이페이지 → "회원 탈퇴" 탭 → 탈퇴 안내 화면 이동
- [ ] 안내 화면: "정말 떠나시는 건가요?" + 삭제 항목 명시 (먹은 이력, 즐겨찾기, 취향 설정, 프로필 정보)
- [ ] "탈퇴하기" 탭 → 최종 확인 팝업: "정말 탈퇴하시겠어요?" [취소] [탈퇴]
- [ ] 취소 → 팝업 닫힘
- [ ] 확인 → 계정 삭제 → 로그인 화면 이동
- [ ] 탈퇴 후 동일 Google 계정으로 재가입 가능 (신규 계정 취급)
- [ ] 탈퇴 처리: 즉시 소프트 삭제 → 30일 뒤 물리 삭제

## 23. 탭 네비게이션

- [ ] 홈 탭 → 식당 상세 → 뒤로 → 홈 탭 복귀 (다른 탭으로 빠지지 않음)
- [ ] 탐색 탭 → 식당 상세 → 뒤로 → 탐색 탭 복귀 (다른 탭으로 빠지지 않음)
- [ ] 이력 탭 → 식당 상세 → 뒤로 → 이력 탭 복귀 (다른 탭으로 빠지지 않음)
- [ ] 마이 탭 → 프로필 편집 → 뒤로 → 마이 탭 복귀 (다른 탭으로 빠지지 않음)
- [ ] 각 탭 독립 스택 유지

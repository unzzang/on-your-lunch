# Gate/Phase 진입 규칙

Stage의 Gate 또는 Phase에 진입할 때 아래 절차를 수행한다.

## 진입 시 체크

1. **해당 프로세스 문서를 먼저 읽는다** — `processes/`에서 해당 Stage의 절차서를 확인.

2. **활동을 3가지로 분류한다:**
   - 🔵 직접 할 것
   - 🟡 PO와 협의가 필요한 것
   - 🟢 전문 에이전트에게 위임할 것

3. **🔵과 🟡을 먼저 수행한 후에 🟢을 진행한다** — 방향 없이 통째로 위임하지 않는다.

4. **DR이 필요한 결정은 반드시 기록한다.**

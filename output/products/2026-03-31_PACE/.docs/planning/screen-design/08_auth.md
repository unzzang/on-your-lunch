# Auth 화면 설계

> 상태: 설계 중
> 디자인 파일: `design/pace-app-auth.pen`

---

## 화면 개요

로그인/회원가입/비밀번호 찾기. 간결하고 빠른 진입이 목표.

## 화면 구조

### 1. 로그인 화면 (기본 진입)

#### 상단
- PACE 로고 (중앙, 큰 사이즈)
- "의지력의 외주화" 태그라인 (Inter 14px, #B3B3B3)

#### 소셜 로그인 (우선 표시)
- [Google로 계속하기] 버튼 (전체 너비, 흰색 배경, 검정 텍스트)
- [Apple로 계속하기] 버튼 (전체 너비, 흰색 배경, 검정 텍스트)
- [카카오로 계속하기] 버튼 (전체 너비, #FEE500 배경, 검정 텍스트)

#### 구분선
- ──── 또는 ──── (Inter 12px, #6A6A6A)

#### 이메일 로그인
- 이메일 입력 (#282828 배경, radius 8px)
- 비밀번호 입력 (#282828 배경, radius 8px)
- [로그인] 버튼 (전체 너비, #1DB954)

#### 하단 링크
- "비밀번호를 잊으셨나요?" (#B3B3B3)
- "계정이 없으신가요? 회원가입" (#1DB954)

### 2. 회원가입 화면

#### 상단
- ← 뒤로 가기
- "회원가입" 제목

#### 입력 폼
- 이름 입력
- 이메일 입력
- 비밀번호 입력
- 비밀번호 확인

#### 약관 동의
- [전체 동의] 체크박스
- [이용약관 동의 (필수)] + "보기" 링크
- [개인정보 처리방침 동의 (필수)] + "보기" 링크
- [마케팅 수신 동의 (선택)]

#### 하단
- [회원가입] 버튼 (#1DB954, 전체 너비)

### 3. 비밀번호 찾기

- ← 뒤로
- "비밀번호 찾기" 제목
- 이메일 입력
- [재설정 링크 보내기] 버튼
- "로그인으로 돌아가기" 링크

## 인터랙션

| 요소 | 동작 | 결과 |
|---|---|---|
| 소셜 로그인 버튼 | 탭 | 해당 소셜 인증 → 온보딩 or Home |
| [로그인] | 탭 | 인증 → 온보딩 or Home |
| [회원가입] | 탭 | 가입 완료 → 온보딩 |
| "비밀번호를 잊으셨나요?" | 탭 | 비밀번호 찾기 화면 |
| "회원가입" 링크 | 탭 | 회원가입 화면 |

## 상태별 분기

| 상태 | 이동 |
|---|---|
| 신규 가입 (소셜/이메일) | → 온보딩 |
| 기존 사용자 로그인 | → Home |
| 온보딩 미완료 사용자 | → 온보딩 (이어서) |

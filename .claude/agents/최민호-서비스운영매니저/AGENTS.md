# 최민호 — 서비스 운영 매니저 (운영팀장)

서비스 안정성과 운영 효율을 책임지는 운영 전문가. 장애 감지부터 해결 확인까지 전 과정을 관리하고, 운영 프로세스를 체계화한다.

## 성격

- 장애 감지 → 에스컬레이션 → 해결 확인까지 끝까지 책임진다.
- 모니터링 대시보드와 알림 체계를 먼저 구축한 뒤 운영에 들어간다.
- 장애 발생 시 원인 분석(포스트모템)을 반드시 수행하고 기록으로 남긴다.

## 담당 범위

- **서비스 모니터링** — 가동률, 응답 시간, 에러율 모니터링 체계 수립
- **장애 대응** — 감지 → 알림 → 대응 → 복구 → 포스트모템 프로세스 운영
- **앱스토어 관리** — 앱 심사 대응, 버전 관리, 스토어 페이지 업데이트
- **배포 관리** — 릴리스 체크리스트, 롤백 절차, 배포 일정 관리
- **운영 SOP** — Standard Operating Procedure 문서화 및 관리
- **서비스 지표 리포트** — DAU, MAU, 크래시율, API 에러율 정기 보고

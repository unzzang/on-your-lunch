# 데이터 분석 + 지표 설계

서비스의 핵심 지표를 설계하고, 데이터 기반 의사결정을 지원한다.

## 작업 기준

### 지표 설계

1. 핵심 지표(North Star Metric) 정의
2. 기능별 KPI 설계 (AARRR 등 프레임워크 활용)
3. 이벤트 트래킹 설계 (이벤트명, 속성, 트리거 조건)
4. 대시보드 설계 (차트 종류, 필터, 갱신 주기)

### 원칙

- 모든 지표에 측정 방법과 기준값을 명시한다
- 이벤트 택소노미는 개발팀과 합의 후 확정한다

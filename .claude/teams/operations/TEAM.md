# 운영팀

## 역할

| 담당 | 에이전트 | 범위 |
|------|---------|------|
| 운영 | 최민호 | 서비스 모니터링, 장애 대응, 앱스토어 관리 |
| CS | 정예림 | 사용자 피드백 수집·분류, 리뷰 대응, 문의 처리 |

## CS-개발 피드백 루프

```
사용자 피드백 (앱스토어/CS/SNS)
    ↓
정예림(CS) — 접수 + 분류
    ├─ 버그 → 진도준(QA) → 개발팀 수정
    ├─ 기능 요청 → AI 파트너 → 기획 백로그
    ├─ 단순 문의 → 직접 답변
    └─ 장애 → 최민호(운영) → 개발팀 긴급 대응
```

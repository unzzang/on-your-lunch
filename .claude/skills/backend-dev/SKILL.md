# 백엔드 API 구현

설계된 기술 스펙과 ERD를 기반으로 백엔드 API를 구현한다.

## 작업 기준

### 구현 순서

1. DB 마이그레이션 (ERD 기반)
2. 엔티티·DTO 정의
3. API 엔드포인트 구현
4. 인증·인가 처리
5. 테스트 작성

### 코드 품질

- 기술 스펙의 API 계약을 정확히 구현한다
- 린트 통과 필수
- 단위 테스트 + e2e 테스트 작성

## 화면 흐름 (Flow)

### 앱 플로우

```
[02_AI_메뉴_추천] "이걸로 투표하기" 탭
        │
        v
[S03-1] 투표방 설정 화면
  ─ 제목 수정, 마감 방식 선택, 후보 확인/삭제
  ─ "투표 시작" 탭
        │
        v
[S03-2] 초대 공유 화면
  ─ 카카오톡 공유 / 링크 복사
  ─ "투표방으로 가기" 탭
        │
        v
[S03-3] 투표 진행 화면
  ─ 메뉴 카드 탭으로 투표, 실시간 결과 확인
  ─ 타이머 만료 / 수동 마감 / 전원 투표 완료
        │
        v
[S03-4] 투표 결과 화면
  ─ 1등 메뉴 확인
  ─ "배달 주문하기" → [04_배달_연동]
  ─ "결과 공유하기" → 카카오톡/링크 공유
```

### 웹 참여 플로우

```
[초대 링크 클릭] (모바일 브라우저)
        │
        v
[S03-5a] 웹 닉네임 입력 화면
  ─ 닉네임 입력 (1~10자)
  ─ "참여하기" 탭
        │
        v
[S03-5b] 웹 투표 진행 화면
  ─ 앱 투표 진행 화면과 동일 레이아웃
  ─ 타이머 만료 / 수동 마감 / 전원 투표 완료
        │
        v
[S03-5c] 웹 투표 결과 화면
  ─ 앱 결과 화면과 동일 레이아웃
  ─ 하단에 앱 설치 유도 배너
```

## 디자인 원칙

배달의민족 최신 개편 스타일을 벤치마크한다.

| 항목 | 기준 |
|------|------|
| 배경색 | 화이트(`#FFFFFF`) 단색 배경. 카드 영역만 연회색(`#F5F5F5`) 배경 |
| 여백 | 섹션 간 상하 여백 32px, 카드 내부 패딩 20px, 좌우 화면 마진 20px |
| 폰트 | 본문 15px Regular, 소제목 17px Bold, 대제목 22px Bold |
| 색상 체계 | 메인 브랜드 컬러(CTA 버튼, 1등 뱃지), 텍스트 `#1A1A1A`, 보조 텍스트 `#8E8E93`, 비활성 `#D1D1D6` |
| 모서리 | 카드 12px radius, 버튼 8px radius, 입력 필드 8px radius |
| 그림자 | 카드에만 `0 2px 8px rgba(0,0,0,0.06)` 적용 |
| 터치 영역 | 모든 터치 가능 요소 최소 44x44px |
| 애니메이션 | 화면 전환 슬라이드 300ms ease-out, 득표수 변경 시 숫자 카운트업 200ms |

## 화면 상세

### S03-1. 투표방 설정 화면 (앱)

**목적:** AI 추천 메뉴가 후보로 미리 채워진 상태에서, 투표 제목과 마감 방식을 설정하고 투표방을 생성한다.

**진입 경로:** 02_AI_메뉴_추천 화면에서 "이걸로 투표하기" 버튼 탭

**레이아웃 (위에서 아래 순서):**

```
┌─────────────────────────────────┐
│ [<] 투표 만들기                  │  ← 네비게이션 바
├─────────────────────────────────┤
│                                 │
│ 투표 제목                        │  ← 레이블 (15px, #8E8E93)
│ ┌─────────────────────────────┐ │
│ │ 오늘의 점심 투표              │ │  ← 텍스트 입력 (17px Bold)
│ │                        24/30│ │  ← 글자 수 카운터 (13px, #8E8E93)
│ └─────────────────────────────┘ │
│                                 │  ← 32px 간격
│ 마감 방식                        │  ← 레이블
│ ┌─────────┐┌─────────┐┌───────┐ │
│ │  5분    ││  10분   ││ 수동  │ │  ← 세그먼트 컨트롤 (3개)
│ │(선택됨) ││         ││  마감  │ │
│ └─────────┘└─────────┘└───────┘ │
│                                 │
│ 5분 후 자동으로 마감돼요          │  ← 설명 문구 (13px, #8E8E93)
│                                 │  ← 32px 간격
│ 투표 후보                        │  ← 레이블 + 후보 수 뱃지 "4개"
│ ┌─────────────────────────────┐ │
│ │ 🍜 베트남 쌀국수       [X]  │ │  ← 후보 카드 1
│ └─────────────────────────────┘ │
│ ┌─────────────────────────────┐ │
│ │ 🍕 페퍼로니 피자       [X]  │ │  ← 후보 카드 2
│ └─────────────────────────────┘ │
│ ┌─────────────────────────────┐ │
│ │ 🥗 연어 포케           [X]  │ │  ← 후보 카드 3
│ └─────────────────────────────┘ │
│ ┌─────────────────────────────┐ │
│ │ 🍛 카레라이스          [X]  │ │  ← 후보 카드 4
│ └─────────────────────────────┘ │
│                                 │
│ AI가 추천한 메뉴예요.            │  ← 안내 문구 (13px, #8E8E93)
│ 원하지 않는 메뉴는 삭제할 수 있어요.│
│                                 │
│ ┌─────────────────────────────┐ │
│ │         투표 시작             │ │  ← CTA 버튼 (#1A1A1A, 흰 텍스트)
│ └─────────────────────────────┘ │
│                                 │  ← 하단 safe area 여백
└─────────────────────────────────┘
```

**구성 요소 상세:**

| 요소 | 사양 |
|------|------|
| 네비게이션 바 | 좌측 뒤로가기 화살표(`<`), 중앙 제목 "투표 만들기" (17px Bold) |
| 투표 제목 입력 | 텍스트 인풋. 기본값 "오늘의 점심 투표". 최대 30자. 하단 우측에 글자 수 카운터 `{현재}/{최대}` 표시. 포커스 시 하단 보더 브랜드 컬러로 변경 |
| 마감 방식 세그먼트 | 3개 선택지: "5분" / "10분" / "수동 마감". 기본 선택 "5분". 선택된 항목: 브랜드 컬러 배경 + 흰 텍스트. 미선택 항목: `#F5F5F5` 배경 + `#1A1A1A` 텍스트. 높이 44px |
| 마감 방식 설명 | "5분" 선택 시 "5분 후 자동으로 마감돼요", "10분" 선택 시 "10분 후 자동으로 마감돼요", "수동 마감" 선택 시 "직접 마감 버튼을 눌러야 마감돼요" |
| 후보 메뉴 카드 | 높이 56px, 좌측 메뉴 이모지 + 메뉴명(15px Regular), 우측 삭제 버튼(`X` 아이콘, 20x20px, `#8E8E93`). 카드 간 간격 8px |
| 후보 수 뱃지 | "투표 후보" 레이블 우측에 `{N}개` 뱃지 (12px, 브랜드 컬러 텍스트, 브랜드 컬러 10% 불투명 배경, 4px radius) |
| "투표 시작" 버튼 | 하단 고정(Sticky). 너비 100%(좌우 마진 20px 제외). 높이 52px. `#1A1A1A` 배경, 흰 텍스트 17px Bold. 12px radius |

**상태 & 인터랙션:**

| 상태 | 표현 |
|------|------|
| 후보 2개 이상 | "투표 시작" 버튼 활성화 (브랜드 컬러) |
| 후보 1개만 남음 | "투표 시작" 버튼 비활성화 (`#D1D1D6` 배경). 후보 목록 하단에 "후보가 2개 이상 필요해요" 문구 표시 (13px, `#FF3B30`) |
| 후보 삭제 탭 | 카드가 좌측으로 슬라이드 아웃 (200ms), 아래 카드가 위로 올라옴 |
| 이미 진행 중인 투표방 존재 | 화면 진입 불가. "진행 중인 투표가 있어요. 마감 후 새로 만들 수 있어요." 토스트(3초) + "기존 투표방 보기" 버튼 |
| 네트워크 오류 (투표방 생성 실패) | "네트워크 연결을 확인해주세요" 토스트(3초) + "투표 시작" 버튼에 재시도 상태 표시 |
| 제목 30자 초과 입력 시도 | 입력 차단. 글자 수 카운터 `30/30` 빨간색(`#FF3B30`)으로 변경 |

### S03-2. 초대 공유 화면 (앱)

**목적:** 투표방 생성 직후 동료에게 초대 링크를 공유한다.

**진입 경로:** S03-1에서 "투표 시작" 탭 후 투표방 생성 완료 시 자동 표시

**표시 방식:** 하프 모달(Half Modal) — 화면 하단 60% 영역을 차지하는 바텀시트. 배경은 딤 처리(`rgba(0,0,0,0.4)`).

**레이아웃:**

```
┌─────────────────────────────────┐
│          (딤 배경 영역)           │  ← 탭하면 모달 닫힘
│                                 │
├─────── ━━━━━━━ ─────────────────┤  ← 드래그 핸들 (40x4px, #D1D1D6, 가운데)
│                                 │
│   동료를 초대해보세요!            │  ← 제목 (22px Bold)
│   링크만 보내면 바로 투표할 수     │
│   있어요.                        │  ← 부제목 (15px, #8E8E93)
│                                 │  ← 24px 간격
│  ┌──────────────────────────┐   │
│  │  📋 초대 링크 복사          │   │  ← 버튼 1 (52px, #F5F5F5 배경)
│  └──────────────────────────┘   │
│                                 │  ← 12px 간격
│  ┌──────────────────────────┐   │
│  │  💬 카카오톡으로 공유       │   │  ← 버튼 2 (52px, 카카오 노란색 #FEE500 배경)
│  └──────────────────────────┘   │
│                                 │  ← 24px 간격
│  ┌──────────────────────────┐   │
│  │      투표방으로 가기        │   │  ← CTA 텍스트 버튼 (브랜드 컬러, 밑줄 없음)
│  └──────────────────────────┘   │
│                                 │  ← 하단 safe area 여백
└─────────────────────────────────┘
```

**구성 요소 상세:**

| 요소 | 사양 |
|------|------|
| 드래그 핸들 | 상단 중앙, 40x4px, `#D1D1D6`, 2px radius. 아래로 드래그하면 모달 닫힘 |
| 제목 | "동료를 초대해보세요!" — 22px Bold, `#1A1A1A` |
| 부제목 | "링크만 보내면 바로 투표할 수 있어요." — 15px Regular, `#8E8E93` |
| "초대 링크 복사" 버튼 | 52px 높이, `#F5F5F5` 배경, `#1A1A1A` 텍스트 15px Medium, 8px radius. 좌측 클립보드 아이콘. 탭 시 "링크가 복사되었어요" 토스트(2초) 표시 + 버튼 텍스트 "복사 완료"로 1초간 변경 |
| "카카오톡으로 공유" 버튼 | 52px 높이, `#FEE500` 배경, `#3C1E1E` 텍스트 15px Medium, 8px radius. 좌측 카카오톡 아이콘. 탭 시 카카오톡 공유 API 호출 |
| "투표방으로 가기" 텍스트 버튼 | 브랜드 컬러 텍스트 15px Medium. 모달 닫힘 + 투표 진행 화면(S03-3)으로 이동 |

**상태 & 인터랙션:**

| 상태 | 표현 |
|------|------|
| 카카오톡 미설치 | "카카오톡으로 공유" 버튼 숨김. "초대 링크 복사" 버튼만 표시 |
| 딤 영역 탭 또는 하단 스와이프 | 모달 닫힘 → S03-3 투표 진행 화면으로 이동 |
| 공유 완료 후 | 모달 자동 닫힘 없음. 사용자가 직접 "투표방으로 가기" 탭 또는 딤 영역 탭으로 닫음 |

### S03-3. 투표 진행 화면 (앱)

**목적:** 실시간으로 메뉴에 투표하고 현재 투표 현황을 확인한다.

**진입 경로:** S03-2에서 "투표방으로 가기" 탭 / 초대 링크 딥링크로 앱 진입 / 홈 화면에서 "진행 중인 투표" 배너 탭

**레이아웃:**

```
┌─────────────────────────────────┐
│ [<]  오늘의 점심 투표    [공유]  │  ← 네비게이션 바
├─────────────────────────────────┤
│                                 │
│  ⏱ 3:42 남음        👥 5명 참여 │  ← 상태 바
│                                 │  ← 16px 간격
│  ┌─────────────────────────────┐│
│  │ 🍜 베트남 쌀국수     🏆 1등  ││  ← 후보 카드 1 (선택됨 상태)
│  │                             ││
│  │ ██████████████░░░░  3표     ││  ← 프로그레스 바 + 득표수
│  │                             ││
│  │ 민수, 지은, 나      +0명     ││  ← 투표자 닉네임
│  │                       ✓ 선택 ││  ← 본인 선택 표시
│  └─────────────────────────────┘│
│                                 │  ← 12px 간격
│  ┌─────────────────────────────┐│
│  │ 🍕 페퍼로니 피자             ││  ← 후보 카드 2
│  │                             ││
│  │ █████████░░░░░░░░░  2표     ││
│  │                             ││
│  │ 현우, 소영           +0명    ││
│  └─────────────────────────────┘│
│                                 │  ← 12px 간격
│  ┌─────────────────────────────┐│
│  │ 🥗 연어 포케                 ││  ← 후보 카드 3
│  │                             ││
│  │ ░░░░░░░░░░░░░░░░░░  0표    ││
│  │                             ││
│  │                              ││
│  └─────────────────────────────┘│
│                                 │
│ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─│  ← 구분선
│ 참여자: 민수, 지은, 현우, 소영, 나│  ← 참여자 목록
│                                 │
│  ┌─────────────────────────────┐│
│  │         지금 마감             ││  ← 수동 마감 버튼 (생성자만 노출)
│  └─────────────────────────────┘│
│                                 │
└─────────────────────────────────┘
```

**구성 요소 상세:**

| 요소 | 사양 |
|------|------|
| 네비게이션 바 | 좌측 뒤로가기, 중앙 투표방 제목(17px Bold, 최대 1줄, 말줄임), 우측 공유 아이콘(S03-2 모달 재표시) |
| 상태 바 | 좌측: 타이머 아이콘 + 남은 시간(17px Bold, 타이머 마감일 때만 표시). 우측: 사람 아이콘 + 참여자 수(15px Regular, `#8E8E93`). 수동 마감 모드에서는 좌측에 "수동 마감" 텍스트 표시 |
| 타이머 색상 변화 | 남은 시간 60초 초과: `#1A1A1A`. 60~31초: `#FF9500`(주황). 30초 이하: `#FF3B30`(빨강) + 타이머 아이콘 펄스 애니메이션 |
| 후보 메뉴 카드 | 높이 가변(내용에 따라). 패딩 20px. 12px radius. 카드 간 간격 12px |
| 카드 — 메뉴명 | 좌측 이모지 + 메뉴명(17px Bold). 1등 메뉴: 우측에 "1등" 뱃지(브랜드 컬러 배경, 흰 텍스트, 12px Bold, 16px radius pill) |

## 화면 상세

### S04-1. 점심 기록 목록 화면

**목적:** 과거 점심 기록을 날짜 역순으로 확인한다.

**진입 경로:** 하단 탭 바 > "기록" 탭

---

#### 레이아웃 구조 (위→아래)

```
┌─────────────────────────────────────┐
│  Status Bar                         │
├─────────────────────────────────────┤
│  앱 바: "점심 기록"                    │  ← 좌측 정렬, 우측 요소 없음
├─────────────────────────────────────┤
│                                     │
│  ┌─────────────────────────────────┐│
│  │ 📅 3월 28일 금요일               ││  ← 날짜 헤더 (섹션 구분)
│  ├─────────────────────────────────┤│
│  │                                 ││
│  │  ☀️ 맑음 · 12°C                 ││  ← 날씨 아이콘 + 텍스트 + 기온
│  │                                 ││
│  │  김치찌개                        ││  ← 메뉴명 (Bold, 큰 텍스트)
│  │                                 ││
│  │  ┌──────┐                       ││
│  │  │ 한식  │                       ││  ← 카테고리 칩
│  │  └──────┘                       ││
│  │                                 ││
│  │  👥 5명 참여                     ││  ← 참여 인원
│  │                                 ││
│  └─────────────────────────────────┘│
│                                     │
│  ┌─────────────────────────────────┐│
│  │ 📅 3월 27일 목요일               ││  ← 다음 기록 카드
│  │  ...                            ││
│  └─────────────────────────────────┘│
│                                     │
│         (스크롤 → 추가 로드)          │  ← cursor 기반 페이지네이션
```

#### 앱 바

| 속성 | 값 |
|------|-----|
| 높이 | 56px |
| 배경색 | `#FFFFFF` |
| 타이틀 | "점심 기록" |
| 타이틀 정렬 | 좌측, 좌측 패딩 20px |
| 타이틀 폰트 | 20px, Bold, `#1E1E1E` |
| 하단 구분선 | 없음 (배민 스타일: 스크롤 시 그림자로 구분) |

#### 날짜 헤더 (섹션 구분)

같은 날짜의 기록은 하나의 섹션으로 묶는다. 현재 MVP에서는 하루 1건이므로 카드마다 날짜 헤더가 붙는다.

| 속성 | 값 |
|------|-----|
| 텍스트 형식 | "M월 D일 E요일" (예: "3월 28일 금요일") |
| 폰트 | 14px, SemiBold, `#9E9E9E` |
| 상단 마진 | 첫 번째 섹션 16px, 이후 섹션 24px |
| 좌측 패딩 | 20px |
| 하단 마진 | 8px |

**오늘 날짜 표기:** 오늘의 기록인 경우 날짜 텍스트 옆에 "오늘" 뱃지 표시.
- 뱃지: 배경 `#2AC1BC`, 텍스트 `#FFFFFF`, 12px, Medium, 패딩 좌우 8px / 상하 2px, border-radius 4px

#### 기록 카드

| 속성 | 값 |
|------|-----|
| 배경색 | `#FFFFFF` |
| 좌우 마진 | 20px |
| 내부 패딩 | 상하 20px, 좌우 16px |
| 모서리 둥글기 | 12px |
| 그림자 | `0 2px 8px rgba(0, 0, 0, 0.06)` |
| 카드 간 간격 | 0px (날짜 헤더의 상단 마진으로 구분) |

**카드 내부 요소 (위→아래):**

1. **날씨 행**
   - 날씨 아이콘 (20x20px) + 날씨 텍스트 + "·" + 기온
   - 폰트: 13px, Regular, `#9E9E9E`
   - 날씨 아이콘 매핑: 맑음=sun, 흐림=cloud, 비=umbrella, 눈=snowflake, 기타=dash
   - 날씨 데이터 `null`인 경우: 아이콘 대신 "—" 표시, 텍스트 "날씨 정보 없음"
   - 하단 마진: 8px

2. **메뉴명**
   - 폰트: 18px, Bold, `#1E1E1E`
   - 최대 1줄, 초과 시 말줄임(...)
   - 하단 마진: 8px

3. **카테고리 칩**
   - 배경: `#F5F5F5`
   - 텍스트: 12px, Medium, `#757575`
   - 패딩: 좌우 10px, 상하 4px
   - border-radius: 6px
   - 하단 마진: 12px
   - 카테고리 마스터 목록 (10개): 한식, 중식, 일식, 양식, 분식, 치킨/피자, 버거, 아시안, 샐러드, 도시락

4. **참여 인원**
   - 사람 아이콘 (16x16px, `#BDBDBD`) + "{N}명 참여"
   - 폰트: 13px, Regular, `#9E9E9E`

**카드 탭 인터랙션:**
- 탭 시 S04-1-D 점심 기록 상세 바텀시트가 올라온다.
- 탭 피드백: 카드 배경색 `#F9F9F9`로 변경 (150ms)

#### 기록 상세 바텀시트 (S04-1-D)

카드를 탭하면 화면 하단에서 올라오는 바텀시트로 상세 정보를 표시한다.

| 속성 | 값 |
|------|-----|
| 바텀시트 높이 | 화면 높이의 50% |
| 배경색 | `#FFFFFF` |
| 상단 핸들바 | 가로 40px, 높이 4px, `#E0E0E0`, border-radius 2px, 상단 마진 12px |
| 모서리 둥글기 | 상단 좌우 16px |
| 배경 딤 | `rgba(0, 0, 0, 0.4)` |
| 닫기 | 핸들바 아래로 드래그, 또는 딤 영역 탭 |

**바텀시트 내부 구성 (위→아래):**

1. **확정 메뉴 영역**
   - "확정 메뉴" 라벨: 12px, Medium, `#9E9E9E`, 하단 마진 4px
   - 메뉴명: 20px, Bold, `#1E1E1E`
   - 카테고리 칩 (카드와 동일 스타일)
   - 구분선: `#F0F0F0`, 1px, 상하 마진 16px

2. **투표 결과 영역**
   - "투표 결과" 라벨: 14px, SemiBold, `#1E1E1E`, 하단 마진 12px
   - 후보 메뉴 리스트 (최대 4개):
     - 각 행: 메뉴명 (15px, Regular, `#424242`) + 우측 득표수 바
     - 득표수 바: 높이 8px, border-radius 4px, 배경 `#F0F0F0`, 채움 `#2AC1BC`
     - 1등 메뉴(확정 메뉴)의 바 색상: `#2AC1BC` (프라이머리)
     - 기타 메뉴의 바 색상: `#E0E0E0`
     - 바 우측에 득표수 텍스트: "N표", 13px, Regular, `#9E9E9E`
     - 행 간 간격: 12px

3. **참여 정보**
   - "참여 인원 {N}명" 텍스트: 13px, Regular, `#9E9E9E`, 상단 마진 16px

#### 빈 상태 (기록 없음)

그룹에 아직 기록이 없는 경우 표시한다.

```
┌─────────────────────────────────────┐
│                                     │
│         (여백 80px)                  │
│                                     │
│      [캘린더 일러스트 64x64px]        │
│                                     │
│     "아직 점심 기록이 없어요"          │
│     "투표가 확정되면 자동으로           │
│      기록돼요"                       │
│                                     │
└─────────────────────────────────────┘
```

| 요소 | 스타일 |
|------|--------|
| 일러스트 | 64x64px, 중앙 정렬, `#E0E0E0` 색상 |
| 주 텍스트 | 16px, SemiBold, `#424242`, 일러스트 하단 16px |
| 보조 텍스트 | 14px, Regular, `#9E9E9E`, 주 텍스트 하단 4px |
| 전체 영역 | 수직 중앙 정렬 (탭바 제외 영역 기준) |

#### 에러 상태

| 상황 | 표시 |
|------|------|
| 네트워크 에러 | "인터넷 연결을 확인해주세요" + "다시 시도" 버튼 |
| 서버 에러 (5xx) | "잠시 후 다시 시도해주세요" + "다시 시도" 버튼 |
| 접근 권한 없음 (403) | "접근 권한이 없어요" (뒤로가기 유도) |

"다시 시도" 버튼: 텍스트 버튼, 14px, SemiBold, `#2AC1BC`, 에러 메시지 하단 16px

#### 페이지네이션

- cursor 기반 무한 스크롤. 1회 요청당 20건.
- 목록 하단 도달 시 자동으로 다음 페이지 요청.
- 로딩 중: 목록 하단에 스피너 표시 (24x24px, `#BDBDBD`).
- 더 이상 데이터 없음: 목록 하단에 "모든 기록을 불러왔어요" 텍스트 (13px, Regular, `#BDBDBD`, 상하 패딩 20px, 중앙 정렬).

#### 메뉴 항목별 상세

| 메뉴 | 우측 요소 | 탭 동작 |
|------|-----------|---------|
| 회사 위치 변경 | ">" 화살표 | 온보딩 위치 설정 화면으로 이동 (수정 모드) |
| 음식 취향 수정 | ">" 화살표 | 온보딩 취향 설정 화면으로 이동 (수정 모드) |

| 버전 정보 | "v{major}.{minor}.{patch}" (13px, Regular, `#9E9E9E`) | 탭 불가 (터치 피드백 없음) |
| 이용약관 | ">" 화살표 | 인앱 웹뷰 (`{도메인}/terms`) |
| 개인정보처리방침 | ">" 화살표 | 인앱 웹뷰 (`{도메인}/privacy`) |

# PACE 기능 목록 (Feature Map)

> 최종 수정: 2026-03-17
> 상태: 브레인스토밍 완료 — QA 리뷰 반영

---

## Auth
- 회원가입
- 로그인
- 소셜 로그인
- 비밀번호 찾기/재설정
- 회원탈퇴 (데이터 삭제 정책 정의 필요)
- 약관 동의 / 개인정보 처리방침

## Onboarding
- 목표 선택 (수능/공무원/자격증 등) → 과목 구성 결정
- D-Day 입력 → 학습 강도 조절
- 하루 가용시간 입력 → 플레이리스트 길이 추천
- **스킵 가능** — 스킵 시 Home에서 인기 플레이리스트부터 노출
- 정상 완료 시 → AI 첫 주 플레이리스트 생성 → Home에 노출
- 3개 질문으로 제한 (이탈 방지)
- 온보딩 정보 재설정: MyPage > 설정 > 학습 목표 변경

## 전체 공통
- 하단 네비게이션 (Home / Search / Library)
- 하단 미니 플레이어 (항상 고정, 스와이프업 → Now Playing 전체화면)
- Now Playing → 스와이프다운 or 뒤로 가기 → 미니 플레이어로 복귀

## Home (메인)

### 섹션 구성 (위→아래 순서)
1. **오늘의 학습 요약** — 순공 시간, 완료 세션 등 간단 요약 카드. 탭 → MyPage 상세 리포트로 이동
2. **빠른 선택** — 최근/자주 쓰는 플레이리스트 2열 그리드 (4~6개). 첫 사용자는 온보딩 추천 or 인기 플레이리스트로 채움
3. **지금 시작하면 딱 좋은** — 현재 시간 + 남은 가용 시간 기반 추천
4. **OO님을 위한 추천** — AI 사용 패턴 기반 맞춤 추천
5. **수험생들이 많이 쓰는** — 전체 인기 플레이리스트
6. **새로 올라온 플레이리스트** — 신규 콘텐츠

### 개인화 전략
| 단계 | 수준 | 시점 |
|---|---|---|
| MVP | 섹션 구성 고정, 섹션 안 콘텐츠만 개인화 (최근 재생, 시간대 추천) | 론칭 시 |
| 이후 | 섹션 순서/노출 여부도 패턴에 따라 변경 (안 쓰는 섹션 숨김 등) | 데이터 축적 후 |

## Search (탐색)

### 첫 화면
- 검색창 + 카테고리 그리드

### 카테고리 분류
| 분류 기준 | 예시 |
|---|---|
| 시험 유형 | 수능, 공무원, 자격증, 어학, 편입 |
| 시간대별 | 1~2시간, 3~4시간, 5시간+, 7시간+ |
| 과목별 | 국어, 수학, 영어, 사회탐구, 과학탐구 |
| 특수 목적 | 벼락치기, 복습 집중, 시험 전날 |
| 인기/신규 | 인기 TOP, 새로 올라온 |

### 검색 대상
- 플레이리스트 (키워드 매칭)
- 제작자 검색은 이후 버전 (프로필 페이지 설계 필요)

### MVP 범위
- 카테고리 탭 → 해당 플레이리스트 목록 (단순 나열)
- 필터/정렬은 이후 버전에서 추가

## Library (내 보관함)

### 목록 구성
- 좋아요/저장한 플레이리스트
- 수정해서 저장한 플레이리스트
- 최근 재생 기록

### 필터 칩
- 전체 | 저장한 | 수정 저장 | 최근 재생

### 정렬
- MVP: 최근 재생순 고정 (정렬 옵션은 이후)

### 관리 기능
- 저장 해제 (좋아요 취소)
- 수정 저장본 삭제

### 수정 저장본 정책
- 원본과 독립된 복사본으로 저장 (원본 업데이트 영향 없음)
- 수정 저장본을 또 수정하면 덮어쓰기
- 한 원본에 대해 수정 저장본 1개만 유지

## Create (생성) — 이후 버전

> MVP에서 제외. "기존 플레이리스트 수정 + 내 보관함 저장"으로 대체.
> Zero-Setup 철학과 충돌, 개발 공수 절감을 위해 이후 버전으로 이동.

### 생성 플로우 (이후)
1. 기본 정보 — 제목, 커버 이미지(선택), 설명(선택), 공개/비공개
2. 세션(트랙) 추가 — "+ 세션 추가" 버튼
3. 미리보기 → 저장

### 세션 시간 입력
- 5분 단위 슬라이더 (최대 2시간)

### 휴식 세션
- 매 세션 사이에 자동 삽입
- 휴식 시간 일괄 수정 가능 (기본 10분)

### 공개/비공개
- 비공개: 본인만 보관함에서 확인
- 공개: 커뮤니티(플레이리스트 목록)에 노출

## 플레이리스트 (핵심 기능)

### 누가 만드는가
| 제작자 | 방식 | MVP |
|---|---|---|
| 운영팀(PACE) | 시험별/시간별 큐레이션, 수기 제작 | O |
| AI | 사용 패턴 분석 → 개인 맞춤 자동 생성 | △ (이후) |
| 사용자 | Create에서 직접 제작 (공개/비공개) | - (이후) |

### 상세 화면 구조 (Spotify 벤치마크)
- 히어로: 커버 이미지 + 제목 + 제작자
- ▶ 플레이 버튼 + 좋아요 + 기타 액션 (셔플 버튼 제외)
- 트랙 리스트: # | 세션명 | 시간
- 하단: "이건 어떠신가요?" 추천 플레이리스트 (추천 기준 필요)

### 사용자 조정 (전부 가능)
- 세션 순서 변경 (드래그)
- 세션 시간 조정 (50분 → 40분)
- 세션 추가/삭제
- 수정한 플레이리스트 → 내 보관함에 별도 저장 (원본과 독립된 복사본)

### 플레이 플로우
```
플레이리스트 선택 → (필요 시) 시간/순서 조정 → ▶ 플레이 → Now Playing 진입
```

### 동시 세션 방지
- 재생 중 다른 플레이리스트 플레이 시 → 확인 팝업
- "진행 중인 학습이 있어요. 종료하고 새로 시작할까요?" [종료하고 시작] [취소]
- 종료된 세션의 진행 데이터는 Fact-Check에 기록

## Now Playing (집중 모드)

### 화면 구성
- 현재 세션 정보 (과목명 + 큰 타이머 + 진행 바)
- 컨트롤: ⏮ 이전 / ⏸ 일시정지 / ⏭ 다음
- 다음 세션 미리보기
- 전체 진행도 (3/8 세션 완료)
- 스와이프업 → 플레이리스트 트랙 리스트 확인
- 스와이프다운 → 미니 플레이어로 복귀

### 집중 모드
- **Option A (완전 통제형) — MVP:** 앱 이탈 시 타이머 자동 일시정지. 복귀 시 "이어서 할까요?" 팝업. 이탈 시간 기록.
- **Option B (자율형) — 이후:** 백그라운드 타이머 유지. 다이내믹 아일랜드/상태바 표시.
- **휴식 세션:** 집중 모드 미적용 — 앱 이탈해도 타이머 계속 진행 (물 마시러 가는 등 정상 행동)

### 15분 초과 미완료 정책
- **트리거:** 앱 이탈 15분 미복귀 또는 일시정지 15분 경과 (둘 다 동일 적용)
- 선택지 팝업 제공:
  - "이어서 하기" → 현재 세션 남은 시간부터 계속
  - "다음 세션으로" → 현재 세션 미완료 처리, 다음 트랙 시작
  - "오늘은 여기까지" → 플레이리스트 종료 → 완료 바텀시트
- 이탈 15분은 Fact-Check에 "딴짓 시간"으로 기록

### 세션 전환
- **자동 전환** — 5초 카운트다운 후 다음 세션 시작
- 전환 시 **진동 + 알림음**으로 사용자에게 알림
- [바로 시작] [스킵] 버튼 제공
- **스킵 = 카운트다운 건너뛰고 바로 다음 세션 시작** (세션 자체를 건너뛰려면 ⏭ 사용)

### 플레이리스트 완료
- 마지막 세션 종료 시 → 완료 바텀시트:
  - 완료 축하 + 간단 요약 (순공 시간, 집중률)
  - "복습 스케줄 잡아드릴까요?" → [1일 후] [3일 후] [7일 후] (복수 선택 가능)
  - [리포트 자세히 보기] [닫기]
- 닫기 → Home으로 이동
- 리포트 자세히 보기 → Fact-Check 상세 (닫으면 Home)

### 미니 플레이어
- Now Playing 나가도 하단에 항상 고정
- 과목명 + 남은 시간 + 일시정지/재생 버튼

### 엣지 케이스
- **전화 수신:** 앱 이탈과 동일 처리 (타이머 일시정지, 15분 초과 시 미완료 팝업)
- **앱 강제 종료/크래시:** 세션 진행 데이터 로컬 저장, 재실행 시 "이어서 할까요?" 제공
- **자정 넘김:** 시작일 기준으로 기록 (22시 시작 → 해당 날짜의 학습으로 집계)
- **네트워크 끊김:** 타이머 로컬 동작, 온라인 복귀 시 서버 동기화

### MVP에서 제외
- 셔플 (학습에 불필요)
- Option B 자율형

## Fact-Check 리포트

### 핵심 지표
- **순공 시간** = 총 타이머 작동 시간 - 일시정지 시간
- **딴짓 시간** = 앱 이탈 시간 + 15분 초과 일시정지
- **집중률** = 순공 시간 / (순공 시간 + 딴짓 시간) × 100
- 과목별 집중률 (프로그레스 바)
- 시간대별 집중도 패턴 (언제 집중 잘 되는지)

### 앱 이탈 추적 범위
- PACE 앱 이탈 시점 / 복귀 시점 → 이탈 총 시간 기록
- 다른 앱 사용 내역은 추적 불가 (iOS 정책 제한)
- "앱을 떠나 있었던 시간: 45분" 수준으로 제공

### AI 한마디 (인사이트)
- MVP: 룰 기반 메시지 (조건별 미리 작성, 20~30개 룰) → 비용 0
- 이후: LLM 호출로 개인화 코멘트 전환
- 예: "수학은 오전에 배치하면 집중도가 올라갈 수 있어요"

### 리포트 단위
| 단위 | MVP | 이후 |
|---|---|---|
| 일간 (오늘 하루) | O | - |
| 주간 (요일별 비교) | O | - |
| 월간 (한 달 패턴) | - | O |
| 과목별 추이 | - | O |

### 접근 경로
- 홈 → 오늘의 학습 요약 카드 탭 → 상세 리포트
- MyPage → 학습 스탯 → 상세 리포트
- 플레이리스트 완료/종료 시 → 바텀시트 → "리포트 자세히 보기"
- 리포트에서 닫기 → Home으로 이동

### AI 자동 스케줄 조정 (피드백 루프) — 이후 버전
- 성취율 낮은 과목 세션 길이 축소
- 집중도 높은 시간대에 어려운 과목 배치
- 복습 누락 과목 자동 재스케줄링

## 복습 스케줄링 (Ebbinghaus)

### MVP — 가볍게 시작
- 플레이리스트 완료 시 → "복습 스케줄 잡아드릴까요?" 제안
- 사용자가 [1일 후] [3일 후] [7일 후] 선택 (복수 선택 가능)
- 해당 날짜에 푸시 알림 → 사용자가 복습 플레이리스트 선택해서 실행

### 이후 버전
- 사용자 마킹 방식 — 세션 끝날 때 "복습 필요" 체크
- 복습 주기 커스텀 (기본값: 1, 3, 7일 / 사용자 변경 가능)
- AI 기반 복습 시점 자동 최적화

## 알림

### 학습 중 (Now Playing 작동 시)
| 알림 | 시점 | 방식 |
|---|---|---|
| 세션 시작 | 새 세션 진입 | 진동 + 알림음 |
| 세션 종료 임박 | 종료 1~2분 전 | 진동 |
| 휴식 시작 | 휴식 세션 진입 | 진동 + 알림음 |
| 휴식 끝나요 | 휴식 종료 1분 전 | 진동 + 알림음 |
| 세션 전환 | 다음 세션 자동 시작 | 진동 + 알림음 |

### 학습 밖 (앱 미사용 시)
| 알림 | 시점 | 예시 | MVP |
|---|---|---|---|
| 모닝콜 | 사용자 지정 시간 | "공부 시작하셔야죠?" | O |
| 학습 리마인더 | 오늘 아직 미시작 | "오늘 아직 시작 전이에요" | O |
| 장기 미접속 | 2일 이상 미접속 | "돌아와요! 잘하고 있었는데" | O |
| 복습 알림 | 복습 예정일 | "오늘 수학 복습일이에요" | O |
| 주간 리포트 | 매주 일요일 | "이번 주 학습 요약 도착" | O |

### 알림 설정
- **전체 ON/OFF** — 디폴트로 최상위에 존재
- **전체 ON일 경우** → 하위 알림별 개별 ON/OFF 제공
  - 모닝콜 (시간 설정 포함)
  - 학습 리마인더
  - 장기 미접속
  - 복습 알림
  - 주간 리포트

## MyPage

### 상단 스탯 요약
- 총 학습 시간
- 연속 학습 일수
- 평균 집중률

### 메뉴
- 학습 스탯 → Fact-Check 리포트 상세
- 설정
- 구독 관리 → 프리미엄 상태, 결제 정보
- 도움말/문의 → FAQ, 고객센터
- 앱 버전 / 오픈소스 라이선스
- 로그아웃

### 설정 항목
- 알림 설정 (전체 ON/OFF + 개별)
- 학습 목표 변경 (온보딩 정보 재설정)
- 계정 관리 (비밀번호 변경, 탈퇴)
- 다크/라이트 모드
- 집중 모드 기본값 (Option A/B — 이후 버전)

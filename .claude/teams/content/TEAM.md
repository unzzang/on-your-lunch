# 콘텐츠팀

## 역할

| 담당 | 에이전트 | 범위 |
|------|---------|------|
| 블로그 | 강서연 | SEO 최적화 글 작성, 기술 블로그, 서비스 소개 |
| 영상 | 류현우 | 유튜브 기획, 스크립트, 편집 가이드 |

## 작업 흐름

```
블로그: 키워드 리서치 → 구조 설계 → 초안 작성 → 검증(송현아) → 발행
영상:   주제 선정 → 스크립트 → 편집 가이드 → 검증(송현아) → PO 촬영/편집 → 발행
```

## 핵심 규칙

- 비주얼(썸네일, 이미지)은 디자인팀 박소율에게 위임
- 콘텐츠 발행 전 검증전문가(송현아)의 팩트 체크 필수

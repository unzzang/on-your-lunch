# 마케팅 프로세스

> 적용 범위: GTM 전략 수립부터 채널 운영까지
> 책임자: PO (의사결정), 한지민 (전략), 이수빈 (채널 운영)
> 선행 조건: Stage 2 Gate 1 완료 (서비스 정의 확정)
> 스킬: `/marketing`

---

## 1. 역할

서비스의 시장 진입 전략을 수립하고, 채널을 운영하며, 데이터 기반으로 성과를 개선한다.

---

## 2. 프로세스

```
Phase 1 → Phase 2 → Phase 3 → Phase 4 (반복)
```

### Phase 1. GTM 전략 수립 (한지민)

**절차:**

1. 타겟 재검증 (`docs/003_research/` 리서치 결과 참조)
2. 채널 선정 + 예산 배분
3. 핵심 메시지 설계
4. 전략 문서 작성 → PO 승인

**산출물:** 마케팅 전략 문서

**완료 기준:**

- [ ] GTM 전략 문서 확정
- [ ] PO 승인 → DR 기록 (`docs/001_decision-making/`)

---

### Phase 2. 콘텐츠 연계

**절차:**

1. 콘텐츠팀(강서연, 류현우)에 마케팅 방향 전달
2. 디자인팀 박소율에 광고 크리에이티브 요청
3. 콘텐츠 일정과 마케팅 일정 동기화

**완료 기준:**

- [ ] 콘텐츠팀에 방향 전달 완료
- [ ] 크리에이티브 요청 완료
- [ ] 일정 동기화 완료

---

### Phase 3. 채널 운영 (이수빈)

**절차:**

1. SNS 콘텐츠 일정 관리 + 발행
2. 커뮤니티 대응
3. 성과 측정 → 데이터팀(톰크루즈)과 협업

**완료 기준:**

- [ ] 채널 운영 개시
- [ ] 성과 측정 체계 구축 (CAC, 전환율, 리텐션 등)

---

### Phase 4. 성과 분석 + 개선 (반복)

**절차:**

1. 채널별 KPI 추적
2. 데이터 기반 채널 최적화
3. 전략 업데이트 → PO 보고

**완료 기준:**

- [ ] 최소 1회 성과 분석 → 개선 사이클 완료
- [ ] PO 보고 완료

---

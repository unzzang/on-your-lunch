# 서술 규칙

모든 에이전트가 문서를 작성할 때 지켜야 하는 규칙.

## 절대 금지

1. **가짜 데이터 금지** — AI가 생성·추정·시뮬레이션한 수치를 문서에 넣지 않는다. 데이터가 없으면 "데이터 없음"으로 표기하고 수집 방법을 제시한다.

2. **모호 표현 금지** — "적절한", "의미 있는", "충분한" 사용 불가. 구체적 수치와 기준으로 대체한다.

## 필수

3. **출처 명시** — 모든 수치에 출처(URL, 보고서명) 명시.

4. **근거 명시** — 모든 판단에 근거. "이렇게 판단한 이유는..."

5. **한국어** — 모든 문서와 소통은 한국어.

## 표 작성 시

- 표 앞에 도입부 1~3문장 (무엇을 보여주는 표인지)
- 표 뒤에 해석 1~3문장 (핵심 시사점)

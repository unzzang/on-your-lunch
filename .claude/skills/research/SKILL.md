# 심층 사전조사 (Deep Research)

주어진 주제에 대한 사전조사를 수행한다. 상세 절차는 `processes/Research_Process.md`를 참조한다.

## 작업 기준

### 5단계 프로세스

```
Phase 0: PO 인터뷰       — 위더가 직접 (동기, 결정 목표, 타겟, 경쟁사)
Phase 1: 조사 설계       — 조사설계가 (KRQ, 가설, 조사 범위)
Phase 2: 4명 병렬 리서치 — 시장분석 + 경쟁분석 + 고객리서치 + 실현가능성
Phase 3: 검증            — 검증전문가 (교차 검증, 팩트체크)
Phase 4: 전략 종합       — 전략가 (종합 보고서 + Go/No-Go 판단)
```

### 규칙

- 위더는 조율자이지 직접 보고서를 작성하지 않는다
- Phase 0 완료 후 Phase 1 진행, 순차적으로 진행한다
- 가짜 데이터 절대 금지 — 출처 없는 수치는 "데이터 없음" 표기

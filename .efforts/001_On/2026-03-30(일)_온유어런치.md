# 온유어런치 (On Your Lunch)

## 개요

강남 직장인의 점심 메뉴 고민을 해결하는 모바일 앱. 회사 위치 + 취향 + 먹은 이력 기반으로 매일 점심 식당 3곳을 추천한다.

- **Stage:** Stage 3~4 (디자인 + 개발 병행)
- **진행 중인 작업:**

  - 백엔드: 프로젝트 세팅 완료 (NestJS + Prisma + 11개 모듈), API 비즈니스 로직 구현 예정

## FC-002. 일간 리포트

### 핵심 지표 카드 (상단)

| 지표 | 산출 공식 | 포맷 |
|---|---|---|
| 순공 시간 | 총 타이머 작동 시간 - 일시정지 시간 | "3시간 24분" |
| 딴짓 시간 | 앱 이탈 시간 + 15분 초과 일시정지 시간 | "45분" |
| 집중률 | 순공 / (순공 + 딴짓) × 100 | "78%" |

### 집중률 특수 케이스
- 순공 + 딴짓 = 0 (학습 안 함): "-" 표시
- 딴짓 시간 0: 집중률 100%
- 소수점: 반올림하여 정수 표시

### 집중률 개념 구분 (중요)

| 구분 | 공식 | 사용 위치 |
|---|---|---|
| **일간 집중률** | 순공 / (순공 + 딴짓) × 100 | Fact-Check 핵심 카드, MyPage 스탯 평균 |
| **세션/플레이리스트 집중률** | 순공 시간 / 배정 총 시간 × 100 | Now Playing 완료 바텀시트, 과목별 프로그레스 바 |

- **세션 집중률의 미완료 처리:** 미완료 세션도 분모는 배정 시간 기준 (예: 50분 배정 → 20분 순공 → 집중률 40%)
- 두 공식은 다른 개념이며 맥락에 따라 구분하여 사용

### 과목별 집중률 (프로그레스 바)
- 각 과목별 순공 시간 / 배정 시간 × 100
- 프로그레스 바: accent 색상 채움
- 과목별 순공 시간 텍스트 표시
- 정렬: 집중률 높은 순

### 시간대별 집중도 패턴 (막대 차트)
- X축: 시간대 (오전/오후/저녁/심야 또는 시간별)
- Y축: 해당 시간대 순공 시간
- MVP: 4구간 (오전 05~12시 / 오후 12~18시 / 저녁 18~22시 / 심야 22~05시)

### 세션 로그 (하단)
- 오늘 플레이한 세션 목록
- 각 세션: 시작 시간 | 세션명 | 순공 시간 | 상태(완료/미완료)
- 시간순 정렬

## FC-003. 주간 리포트

### 요일별 학습 시간 (막대 차트)
- X축: 월 ~ 일
- Y축: 순공 시간
- 오늘 해당 막대 accent 색상 강조

### 주간 합계
- 총 순공 시간
- 총 딴짓 시간
- 주간 평균 집중률
- 학습한 일수 / 7

### 요일별 비교
- 가장 많이 공부한 요일 하이라이트
- "수요일에 가장 집중을 잘 했어요" 문구

## FC-004. AI 한마디 (인사이트)

### MVP 구현 — 룰 기반 메시지

조건별 미리 작성된 메시지를 표시. LLM 호출 없음 (비용 0).

### 룰 예시

| 조건 | 메시지 |
|---|---|
| 집중률 ≥ 90% | "대단해요! 오늘 집중력이 최고였어요 👏" |
| 집중률 70~89% | "잘하고 있어요! 조금만 더 집중하면 완벽해요" |
| 집중률 50~69% | "나쁘지 않았어요. 내일은 조금 더 화이팅!" |
| 집중률 < 50% | "오늘은 힘들었죠? 내일 다시 시작하면 돼요" |
| 연속 학습 7일 달성 | "일주일 연속 학습 달성! 🔥 대단한 끈기예요" |
| 특정 과목 집중률 < 40% | "{과목}은 시간을 줄이거나 시간대를 바꿔보면 어떨까요?" |
| 오전 집중도 > 오후 | "오전에 집중이 잘 되시네요! 어려운 과목을 오전에 배치해보세요" |
| 딴짓 시간 > 순공 시간 | "오늘은 좀 쉬었네요. 내일은 짧은 플레이리스트로 시작해볼까요?" |

### 표시 위치
- 리포트 상단, 핵심 지표 카드 바로 아래
- 1일 1메시지 (가장 우선순위 높은 룰 1개만)
- 우선순위: 달성 축하 > 과목별 인사이트 > 일반 격려

### 이후 버전
- LLM 호출로 개인화 코멘트 생성
- 사용자의 학습 패턴 전체를 입력하여 맞춤 조언

## FC-005. 앱 이탈 추적 범위

### 추적 가능
- PACE 앱 이탈 시점 (timestamp)
- PACE 앱 복귀 시점 (timestamp)
- 이탈 총 시간 = 복귀 - 이탈

### 추적 불가 (OS 정책 제한)
- 이탈 중 사용한 다른 앱 정보
- 다른 앱 사용 시간 세부 내역

### 사용자에게 제공하는 정보
- "앱을 떠나 있었던 시간: 45분" 수준
- "어떤 앱을 썼는지"는 절대 표시하지 않음

## FC-007. 리포트 탭 전환

### UI
- 상단 탭: [일간] [주간]
- 기본 탭: 일간
- 탭 전환 시 데이터 로드 (캐시 활용)

### 일간 → 날짜 이동
- 좌우 스와이프 또는 < > 화살표로 날짜 이동
- 최대 과거: 가입일
- 미래 이동 불가 (오늘까지만)

### 주간 → 주 이동
- 좌우 스와이프 또는 < > 화살표로 주 이동
- "3월 2주차 (3.10 ~ 3.16)" 형식 표시

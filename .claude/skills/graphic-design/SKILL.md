# 콘텐츠 비주얼 제작

서비스의 마케팅·콘텐츠용 비주얼을 제작한다. 썸네일, 배너, 블로그 삽입 이미지, 광고 크리에이티브 등.

## 작업 기준

### 브랜드 일관성

- 디자인 시스템(컬러, 폰트, 로고)을 반드시 준수한다
- 김지수(UI/UX 디자이너)가 확정한 디자인 시스템을 기준으로 작업한다

### 사이즈 규격

각 채널별 규격을 준수한다:

- **블로그 썸네일** — 채널별 권장 사이즈
- **SNS 배너** — 플랫폼별 권장 사이즈
- **광고 크리에이티브** — 매체별 규격

### 도구

Figma MCP를 통해 직접 작업한다.

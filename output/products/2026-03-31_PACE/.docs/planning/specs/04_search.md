## SRC-001. 탐색 메인 화면

### 구성
- 상단: "탐색하기" 제목
- 검색창 (플레이스홀더: "플레이리스트, 과목 검색")
- 카테고리 그리드 (2열)

### 카테고리 목록

| ID | 카테고리 | 분류 기준 | MVP |
|---|---|---|---|
| cat_suneung | 수능 | 시험 유형 | O |
| cat_civil | 공무원 | 시험 유형 | O |
| cat_cert | 자격증 | 시험 유형 | O |
| cat_lang | 어학 | 시험 유형 | O |
| cat_transfer | 편입 | 시험 유형 | O |
| cat_1_2h | 1~2시간 | 시간대별 | O |
| cat_3_4h | 3~4시간 | 시간대별 | O |
| cat_5h_plus | 5시간+ | 시간대별 | O |
| cat_popular | 인기 TOP | 인기/신규 | O |
| cat_new | 새로 올라온 | 인기/신규 | O |

### 카테고리 매칭 로직

- 시험 유형 카테고리: 플레이리스트의 `exam_type` 값과 매칭 (카테고리 ID → exam_type 매핑 아래 참조)
- 시간대별 카테고리: 플레이리스트 `total_duration` 범위로 매칭
  - 1~2시간: 60분 ≤ total_duration ≤ 120분
  - 3~4시간: 180분 ≤ total_duration ≤ 240분
  - 5시간+: total_duration ≥ 300분
- 인기 TOP: 최근 7일 재생 수 상위 순
- 새로 올라온: 최근 30일 내 등록, 등록일 내림차순

### 카테고리 ID → exam_type 매핑

| 카테고리 ID | exam_type 값 |
|---|---|
| cat_suneung | `suneung` |
| cat_civil | `civil_service` |
| cat_cert | `certification` |
| cat_lang | `language` |
| cat_transfer | `transfer` |

> `other` 타입은 별도 카테고리 없음. 키워드 검색으로만 탐색 가능.

## SRC-002. 검색 기능

### 트리거
- 검색창 탭 → 검색 모드 진입 (키보드 올라옴)

### 검색 대상
- 플레이리스트 제목
- 플레이리스트 설명
- 과목 태그

### 검색 로직
- 입력 시작 후 300ms 디바운스 → 서버 검색 요청
- 최소 검색어: 1자 (한글 기준)
- 검색 결과: 관련도순 정렬 (제목 매칭 > 설명 매칭 > 태그 매칭)

### 검색 결과 표시
- 세로 리스트
- 각 행: 커버 이미지(48x48) + 제목 + 제작자 + 총 시간
- 행 탭 → 플레이리스트 상세 이동

### 검색 결과 없음
- "검색 결과가 없어요"
- 추천 키워드 칩: 인기 검색어 5개 (서버에서 제공)
- 키워드 칩 탭 → 해당 키워드로 자동 검색

### 최근 검색어
- 검색 모드 진입 시 (입력 전): 최근 검색어 목록 표시
- 최대 10개, 최신순
- 개별 삭제 (X 버튼) + 전체 삭제
- 로컬 저장

## SRC-003. 카테고리 상세

### 트리거
- 카테고리 카드 탭

### 화면 구성
- 상단: ← 뒤로 + 카테고리명
- 플레이리스트 목록 (세로 리스트, 단순 나열)
- 각 행: 커버 이미지(56x56) + 제목 + 제작자 + 총 시간 + 세션 수

### 정렬
- MVP: 고정 정렬 (인기순)
- 이후: 정렬 옵션 (인기순 / 최신순 / 시간순)

### 페이지네이션
- 무한 스크롤 (20개씩 로드)
- 더 이상 데이터 없으면 로딩 중단

### 빈 상태
- "아직 이 카테고리에 플레이리스트가 없어요"

# 이수빈 — SNS/커뮤니티 매니저

SNS 채널 운영과 커뮤니티 관리 전문가. 브랜드 보이스를 일관되게 유지하면서 사용자와 직접 소통하는 접점을 담당한다.

## 성격

- 톤앤매너 일관성을 최우선으로 지킨다. 채널이 달라도 브랜드 보이스는 하나다.
- 트렌드 민감도가 높고, 밈/시즌 이슈를 빠르게 콘텐츠에 반영한다.
- 부정 여론은 확산 전에 감지하고, 대응 시나리오를 미리 준비한다.

## 담당 범위

- **SNS 운영** — 인스타그램, 트위터/X, 틱톡 채널 관리
- **콘텐츠 캘린더** — SNS 콘텐츠 발행 일정 수립 및 관리
- **커뮤니티 관리** — 댓글, DM, 멘션 대응 (24시간 이내 응답 원칙)
- **브랜드 보이스** — 톤앤매너 가이드 수립 및 유지
- **여론 모니터링** — 사용자 반응 추적 및 여론 리포트
- **인플루언서 협업** — 크리에이터 발굴 및 협업 조율

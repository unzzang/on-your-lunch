## NOTI-001. 학습 중 알림 (Now Playing 작동 시)

앱 내 알림으로, OS 푸시가 아닌 앱 내부 진동/알림음.

| ID | 알림 | 시점 | 방식 | 조건 |
|---|---|---|---|---|
| NOTI-001-A | 세션 시작 | 새 세션 진입 시 | 진동(짧음) + 알림음 | 항상 |
| NOTI-001-B | 세션 종료 임박 | 세션 종료 1분 전 | 진동(짧음) | 학습 세션만 |
| NOTI-001-C | 휴식 시작 | 휴식 세션 진입 시 | 진동(짧음) + 알림음 | 항상 |
| NOTI-001-D | 휴식 끝나요 | 휴식 종료 1분 전 | 진동(짧음) + 알림음 | 항상 |
| NOTI-001-E | 세션 전환 | 다음 세션 자동 시작 시 | 진동(길게) + 알림음 | 항상 |

### 알림음 정책
- 기본 알림음 1종 제공 (부드러운 톤)
- 무음 모드: 진동만 동작
- 방해 금지 모드: 알림 미발생 (OS 정책 따름)

## NOTI-002. 학습 밖 알림 (OS 푸시)

앱 미사용 시 OS 푸시 알림으로 전송.

| ID | 알림 | 시점 | 메시지 예시 | MVP |
|---|---|---|---|---|
| NOTI-002-A | 모닝콜 | 사용자 설정 시간 | "좋은 아침이에요! 오늘도 시작해볼까요?" | O |
| NOTI-002-B | 학습 리마인더 | 설정 시간 + 2시간 경과 & 미학습 | "오늘 아직 시작 전이에요. 짧은 것부터 해볼까요?" | O |
| NOTI-002-C | 장기 미접속 | 2일 이상 미접속 | "돌아와요! 잘하고 있었는데 🙂" | O |
| NOTI-002-D | 복습 알림 | 복습 예정일 오전 9시 | "오늘 수학 복습일이에요! 잊지 마세요" | O |
| NOTI-002-E | 주간 리포트 | 매주 일요일 20시 | "이번 주 학습 요약이 도착했어요 📊" | O |

### 모닝콜 상세 (NOTI-002-A)
- 설정: MyPage > 설정 > 알림 설정 > 모닝콜 시간 설정
- 시간 피커 (30분 단위, 05:00 ~ 12:00)
- 기본값: 08:00
- 매일 반복 (요일 선택은 이후 버전)

### 학습 리마인더 상세 (NOTI-002-B)
- 모닝콜 시간 + 2시간 경과 시점에 학습 기록 확인
- 오늘 학습 기록 있으면 → 미발송
- 오늘 학습 기록 없으면 → 푸시 발송
- 하루 1회만 발송

### 장기 미접속 상세 (NOTI-002-C)
- 마지막 앱 접속 기준
- 2일 미접속 → 1차 푸시
- 5일 미접속 → 2차 푸시 (다른 메시지)
- 7일 이상 → 주 1회 푸시 (최대 4주, 이후 중단)
- 메시지 풀: 5~10개 랜덤 순환

### 복습 알림 상세 (NOTI-002-D)
- 복습 스케줄 예약 시 서버에 알림 예약
- 발송 시간: 예약일 오전 9시 (고정, 이후 사용자 설정 가능)
- 알림 탭 → 앱 실행 → 해당 플레이리스트 상세 화면

### 주간 리포트 상세 (NOTI-002-E)
- 매주 일요일 20:00 발송
- 알림 탭 → 앱 실행 → Fact-Check 주간 리포트

## NOTI-005. 알림 설정 화면

### 구성

```
[전체 알림 ON/OFF] ← 마스터 토글
  ├ 모닝콜 [ON/OFF] + 시간 설정
  ├ 학습 리마인더 [ON/OFF]
  ├ 장기 미접속 알림 [ON/OFF]
  ├ 복습 알림 [ON/OFF]
  └ 주간 리포트 [ON/OFF]
```

### 마스터 토글 로직
- OFF → 하위 항목 전부 비활성화 (dimmed, 조작 불가)
- ON → 하위 항목 개별 ON/OFF 가능
- 마스터 OFF → ON 전환 시 → 하위 항목 이전 상태 복원

### 기본값 (첫 설치 시)
- 전체: ON
- 모닝콜: ON (08:00)
- 학습 리마인더: ON
- 장기 미접속: ON
- 복습 알림: ON
- 주간 리포트: ON

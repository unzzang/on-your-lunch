# 프론트엔드 구현

모바일 앱과 PC 웹의 프론트엔드를 구현한다. 디자인 핸드오프를 기반으로 1:1 구현을 목표로 한다.

## 작업 기준

### 디자인 충실도

- 디자인 시스템의 토큰(컬러, 타이포, 간격)을 코드에 정확히 반영한다
- 4가지 상태(기본, 빈 상태, 로딩, 에러)를 모두 구현한다

### 코드 품질

- 공유 타입(`shared-types/`)을 사용한다
- 린트 통과 필수 (`pnpm lint`)
- 테스트 작성 후 통과 확인 (`pnpm test`)

### 플랫폼

- **모바일** — React Native (Expo)
- **PC 웹** — Next.js (App Router)

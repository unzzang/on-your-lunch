# 유튜브 기획 + 스크립트

유튜브 영상을 기획하고 스크립트를 작성한다.

## 작업 기준

### 기획

- 영상 주제 선정 (타겟 시청자, 검색 키워드 고려)
- 영상 구조 설계 (훅, 본문, CTA)
- 예상 러닝타임 설정

### 스크립트

- 구어체로 자연스럽게 작성
- 타임라인별 화면 지시 포함
- 자막 텍스트 별도 표기

### 역할 분담

- **스크립트 작성** — 류현우 담당
- **썸네일** — 박소율(그래픽 디자이너)에게 요청
- **촬영/편집** — PO 담당

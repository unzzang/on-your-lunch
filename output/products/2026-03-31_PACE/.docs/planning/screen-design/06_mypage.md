# MyPage 화면 설계

> 상태: 설계 중
> 디자인 파일: `design/pace-app-mypage.pen`

---

## 화면 개요

프로필, 학습 스탯 요약, 설정 등 사용자 관련 기능 허브. Home 상단 프로필 아이콘 탭으로 진입.

## 진입 경로
- Home > 프로필 아이콘 (우측 상단)
- Home > 오늘의 학습 요약 카드 탭 → 학습 스탯 상세

## 화면 구조 (위→아래)

### 상단 바
- 좌측: ← 뒤로 가기
- 중앙: "마이페이지"

### 프로필 영역
- 프로필 아바타 (64px 원형, 중앙)
- 이름 (Inter 20px bold, 흰색)
- 가입일 or 시험 유형 (Inter 14px, #B3B3B3)

### 스탯 요약 카드
- 3열 가로 배치 (배경 #282828, radius 8px, padding 16px)
- 총 학습 시간: "142시간" (Inter 24px bold)
- 연속 학습: "12일 🔥" (Inter 24px bold)
- 평균 집중률: "68%" (Inter 24px bold)
- 각 항목 위에 라벨 (Inter 12px, #B3B3B3)

### 메뉴 리스트
- 각 행: 아이콘(20px) + 메뉴명 + > 화살표
- 행 높이: 56px, 구분선(#282828)

메뉴 항목:
1. 📊 학습 스탯 → Fact-Check 리포트 상세
2. ⚙️ 설정 → 설정 화면
3. 💳 구독 관리 → 구독 상태/결제 정보
4. ❓ 도움말/문의 → FAQ, 고객센터
5. 📋 앱 정보 → 버전, 오픈소스 라이선스
6. 🚪 로그아웃 (빨간색 텍스트, 하단 배치)

### 하단 고정
- 미니 플레이어 (재생 중일 때)

## 인터랙션

| 요소 | 동작 | 결과 |
|---|---|---|
| ← 뒤로 | 탭 | Home으로 복귀 |
| 스탯 요약 카드 | 탭 | 학습 스탯 상세 (Fact-Check) |
| 메뉴 행 | 탭 | 해당 화면으로 이동 |
| 로그아웃 | 탭 | 확인 팝업 → 로그인 화면 |

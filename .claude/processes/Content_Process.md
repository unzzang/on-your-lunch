# 콘텐츠 제작 프로세스

> 적용 범위: 블로그 글, 영상 콘텐츠 제작
> 책임자: PO (최종 리뷰), 강서연 (블로그), 류현우 (영상)
> 선행 조건: 서비스 정의 완료 (Stage 2 Gate 1 이후 언제든)

---

## 1. 역할

서비스의 인지도와 신뢰를 높이기 위한 콘텐츠를 기획하고 제작한다. Stage와 독립적으로 운영 가능.

## 2. 프로세스

### 블로그

```
Step 1 → Step 2 → Step 3 → Step 4 → Step 5
```

| Step | 활동 | 담당 |
|------|------|------|
| 1 | 키워드 리서치 | 강서연 |
| 2 | 구조 설계 + 초안 작성 | 강서연 |
| 3 | 비주얼 요청 (썸네일, 삽화) | 박소율 |
| 4 | 팩트체크 검증 | 송현아 |
| 5 | PO 리뷰 → 발행 | PO |

### 영상

```
Step 1 → Step 2 → Step 3 → Step 4 → Step 5
```

| Step | 활동 | 담당 |
|------|------|------|
| 1 | 주제 선정 + 기획 | 류현우 |
| 2 | 스크립트 작성 | 류현우 |
| 3 | 썸네일 요청 | 박소율 |
| 4 | 팩트체크 검증 | 송현아 |
| 5 | PO 리뷰 → PO 촬영/편집 → 발행 | PO |

**운영 방식:** PO가 강서연/류현우에게 직접 지시. 마케팅팀(한지민)과 주제/일정 조율.

## 3. 완료 기준

- [ ] 콘텐츠 초안 작성 완료
- [ ] 팩트체크 통과 (송현아)
- [ ] PO 리뷰 통과
- [ ] 발행 완료

## NP-001. Now Playing 진입

### 진입 경로
- 플레이리스트 상세 > ▶ 플레이 버튼
- 플레이리스트 상세 > 트랙 탭 (해당 세션부터)
- 미니 플레이어 탭 또는 스와이프업

### 화면 전환
- 전체화면 모달 (하단에서 올라오는 트랜지션)
- 스와이프다운 or ∨ 버튼 → 미니 플레이어로 복귀

## NP-002. 타이머 동작

### 타이머 표시
- 큰 타이머: 남은 시간 (카운트다운, "37:24" 형식)
- 진행 바: 경과 시간 / 전체 시간
- 진행 바 좌측: 경과 시간 ("12:36")
- 진행 바 우측: 전체 시간 ("50:00")

### 타이머 정밀도
- 1초 단위 업데이트
- 백그라운드에서도 시스템 타이머 기반 정확한 시간 유지

### 진행 바 인터랙션
- MVP: 드래그 불가 (표시 전용)
- 이후: 드래그로 시간 탐색 (선택적)

## NP-003. 컨트롤

| 버튼 | 동작 | 조건 |
|---|---|---|
| ⏸ 일시정지 | 타이머 멈춤, 아이콘 ▶로 변경 | 재생 중일 때 |
| ▶ 재생 | 타이머 재개, 아이콘 ⏸로 변경 | 일시정지 중일 때 |
| ⏮ 이전 세션 | 이전 트랙으로 이동 | 첫 세션이면 비활성화 (dimmed) |
| ⏭ 다음 세션 | 다음 트랙으로 이동 | 마지막 세션이면 비활성화 |

### ⏮ 이전 세션 상세 로직
- 현재 세션 경과 5초 이상 → 현재 세션 처음으로 리셋
- 현재 세션 경과 5초 미만 → 진짜 이전 세션으로 이동
- 이전 세션의 남은 시간 = 전체 시간 (처음부터 다시)

### ⏭ 다음 세션 상세 로직
- 현재 세션 → 미완료 처리 (Fact-Check에 기록)
- 다음 세션 시작 (전환 카운트다운 없이 즉시)

## NP-004. 집중 모드 (Option A — 완전 통제형, MVP)

### 앱 이탈 감지
- 사용자가 PACE 앱을 떠남 (홈으로 나가기, 다른 앱 전환, 앱 스위처)
- **학습 세션:** 타이머 자동 일시정지 + 이탈 시간 기록 시작
- **휴식 세션:** 타이머 계속 진행 (이탈 감지하지 않음)

### 30초 버퍼 정책 (iOS inactive 대응)
- 제어 센터, 알림 패널 등으로 앱이 일시적으로 inactive 상태가 된 경우
- **30초 이내 복귀:** 이탈로 처리하지 않음. 타이머 계속 진행, 이탈 시간 미기록
- **30초 초과:** 정식 이탈로 처리 → 타이머 정지 + 이탈 시간 기록 시작
- 적용 대상: iOS/Android 공통 (알림 패널, 제어 센터, 잠금 화면 등)

### 앱 복귀 시 (30초 초과 이탈)
- 이탈 시간 15분 미만:
  - 팝업: "이어서 할까요?" [이어서 하기]
  - [이어서 하기] → 타이머 재개
- 이탈 시간 15분 이상:
  - NP-006 (15분 초과 미완료 정책) 적용

### 이탈 시간 기록
- 이탈 시작 시각 + 복귀 시각 → 이탈 시간 산출 (30초 버퍼 초과분만 기록)
- Fact-Check "딴짓 시간"에 합산

## NP-005. 일시정지

### 트리거
- ⏸ 일시정지 버튼 탭

### 화면 변화
- 타이머 멈춤
- ⏸ → ▶ 아이콘 변경
- "일시정지 중" 텍스트 표시
- 일시정지 경과 시간 별도 카운팅 (UI에 미표시, 내부 기록용)

### 15분 경과 시
- NP-006 적용 (앱 이탈 15분 초과와 동일 처리)

## NP-006. 15분 초과 미완료 정책

### 트리거 조건 (둘 중 하나)
1. 앱 이탈 15분 이상 미복귀
2. 일시정지 15분 이상 경과

### 복귀 시 팝업

| 선택지 | 동작 |
|---|---|
| "이어서 하기" | 현재 세션 남은 시간부터 타이머 재개 |
| "다음 세션으로" | 현재 세션 미완료 처리 → 다음 트랙 시작 |
| "오늘은 여기까지" | 플레이리스트 종료 → NP-009 완료 바텀시트 |

### 데이터 기록
- 15분 초과 이탈 시간 → Fact-Check "딴짓 시간"에 기록
- 현재 세션 → 완료 여부에 따라 "완료" 또는 "미완료" 태깅

## NP-007. 세션 전환

### 트리거
- 현재 세션 타이머가 0에 도달

### 전환 플로우

1. 세션 완료 표시: "수학 미적분 완료!" (1초)
2. 5초 카운트다운: "다음 세션까지 5... 4... 3... 2... 1..."
3. 카운트다운 중 버튼:
   - [바로 시작] → 카운트다운 건너뛰고 다음 세션 즉시 시작
   - [건너뛰기] → 다음 세션도 스킵 → 그 다음 세션의 카운트다운
4. 카운트다운 완료 → 다음 세션 자동 시작

### 전환 알림
- 진동 + 알림음 (설정에서 ON/OFF 가능)
- 학습→휴식, 휴식→학습 전환 시 모두 적용

### 휴식 → 학습 전환
- 휴식 종료 1분 전 사전 알림 (진동)
- 휴식 종료 → 학습 세션 카운트다운 → 학습 시작

## NP-008. 스와이프업 — 트랙 리스트 보기

### 트리거
- Now Playing 화면에서 위로 스와이프

### 화면
- 바텀시트로 트랙 리스트 표시 (전체 높이의 70%)
- 현재 재생 트랙 하이라이트 (accent 색상)
- 완료 트랙: 체크마크 + 흐린 텍스트
- 미완료 트랙: 일반 표시
- 트랙 탭 → 해당 세션으로 즉시 이동 (현재 세션 미완료 처리)

---

## NP-009. 플레이리스트 완료

### 트리거
- 마지막 세션 타이머 0 도달
- "오늘은 여기까지" 선택 (중도 종료)

### 완료 바텀시트 구성

1. **축하 메시지:**
   - 전체 완료: "오늘도 해냈어요! 🎉"
   - 중도 종료: "여기까지도 대단해요! 💪"

2. **간단 요약:**
   - 순공 시간: "2시간 45분"
   - 완료 세션: "5/8 세션"
   - 집중률: "78%" — **세션 집중률** (순공 시간 / 배정 총 시간 × 100, FC-002 참조)

3. **복습 제안:**
   - "복습 스케줄 잡아드릴까요?"
   - [1일 후] [3일 후] [7일 후] — 복수 선택 가능
   - 선택한 날짜에 복습 알림 예약

4. **하단 버튼:**
   - [리포트 자세히 보기] → Fact-Check 상세 화면
   - [닫기] → Home 이동

### 데이터 저장
- 세션별 시작/종료 시간, 경과 시간, 이탈 시간
- 플레이리스트 전체 순공 시간, 집중률
- 서버 즉시 동기화 (오프라인 시 로컬 저장 → 온라인 복귀 시 동기화)

## NP-010. 엣지 케이스

### 전화 수신
- 앱 이탈과 동일 처리 (타이머 일시정지, 이탈 시간 기록)
- 통화 종료 후 복귀 → NP-004 복귀 로직 적용

### 앱 강제 종료 / 크래시
- 세션 진행 데이터 1분 단위 로컬 저장 (중간 저장)
- 앱 재실행 시:
  - 마지막 저장 시점 확인
  - 30분 이내: "이어서 할까요?" 팝업 → [이어서] / [새로 시작]
  - 30분 초과: 해당 세션 미완료 처리, Home으로 이동

### 자정 넘김
- 시작일 기준으로 기록 (22시 시작 → 해당 날짜의 학습)
- 타이머는 연속 동작 (자정에 끊기지 않음)

### 네트워크 끊김
- 타이머 로컬 동작 (서버 의존 없음)
- 데이터 로컬 축적 → 온라인 복귀 시 서버 동기화
- 동기화 실패 시 재시도 (최대 3회, 이후 다음 앱 실행 시 재시도)

### 배터리 부족
- 10% 이하: "배터리가 부족해요. 충전하면서 공부하세요!" 토스트 (1회)
- 타이머 동작에 영향 없음

## NP-012. 진행도 인디케이터

### 표시

- 점 인디케이터: ● 완료(accent) / ○ 미완료(muted) / ◉ 현재(accent+테두리)

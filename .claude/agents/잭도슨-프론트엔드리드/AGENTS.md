# 잭도슨 — 프론트엔드 리드

모바일 앱과 PC 웹의 프론트엔드를 구현하는 프론트엔드 개발 리더. 화면 설계 문서와 디자인 시안을 코드로 구현하고, 프론트엔드 아키텍처를 관리한다.

## 성격

- 디자인 시안과 1:1 피델리티를 추구한다. 디자이너(김지수)와 긴밀히 소통한다.
- 상태 관리 구조를 깔끔하게 유지한다. 불필요한 전역 상태를 만들지 않는다.
- 성능(로딩 속도, 렌더링 최적화)을 항상 의식하며 구현한다.

## 담당 범위

- **모바일 앱 구현** — 화면 컴포넌트, 네비게이션, 애니메이션
- **PC 웹 구현** — 페이지, 레이아웃, 반응형 처리
- **프론트엔드 아키텍처** — 폴더 구조, 상태 관리 패턴, API 연동 레이어
- **공유 컴포넌트** — 모바일/웹 공통 타입, 유틸 관리
- **프론트엔드 테스트** — 컴포넌트 테스트, 스냅샷 테스트
- **디자인 토큰 적용** — 디자인 시스템 토큰을 코드에 반영

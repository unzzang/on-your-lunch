# 관리자 웹 + 프론트 인프라

관리자 웹 애플리케이션을 구현하고, 프론트엔드 전체의 빌드 환경을 관리한다.

## 작업 기준

### 관리자 웹

- 관리자 전용 기능(사용자 관리, 콘텐츠 관리, 대시보드)을 구현한다
- 관리자 권한 체계를 프론트에서 처리한다

### 빌드 환경

- 모노레포 빌드 설정, CI/CD 파이프라인 관리
- 프론트엔드 공통 패키지 관리
- 개발 환경 표준화

## 협업

- **잭도슨(프론트엔드)**과 공통 컴포넌트·빌드 환경 공유
- **이인수(백엔드 개발자)**와 관리자 API 연동
- **톰크루즈(데이터)**와 대시보드 데이터 연동
- **진도준(QA)**의 버그 리포트 반영

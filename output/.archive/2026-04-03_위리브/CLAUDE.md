# 위리브 (WeLive)

아파트 관리 플랫폼. 입주민은 공지사항 확인과 투표 참여, 관리사무소는 여러 단지를 하나의 대시보드로 통합 관리한다.

- **타겟:** 수도권 중대형 아파트(500세대 이상) 입주민 + 위탁관리업체
- **차별점:** 관리사무소가 여러 단지를 하나의 대시보드로 관리
- **플랫폼:** 입주민용 모바일 앱 (iOS/Android) + 관리자용 웹 대시보드
- **UX 벤치마크:** 없음

---

## 핵심 기능

### 입주민 (모바일 앱)
- 공지사항 확인
- 투표 참여

### 관리자 (웹 대시보드)
- 다단지 통합 관리
- 공지사항 등록/관리
- 투표 생성/결과 확인

# SNS 채널 운영 + 커뮤니티 관리

서비스의 SNS 채널을 운영하고 사용자 커뮤니티를 관리한다.

## 작업 기준

### 채널 운영

- 플랫폼별 콘텐츠 포맷 최적화 (인스타 카드뉴스, 트위터 스레드 등)
- 발행 스케줄 관리
- 톤앤매너 일관성 유지

### 커뮤니티 관리

- 댓글·DM 대응 가이드라인 준수
- 사용자 피드백 수집 → 정예림(CS)에게 전달
- 이슈 발생 시 즉시 보고

### 원칙

- 브랜드 가이드라인 준수
- 민감한 대응은 PO 확인 후 처리

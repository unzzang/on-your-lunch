# 리서치팀

## 역할

서비스 아이디어의 **시장성, 경쟁 환경, 고객 니즈, 실현 가능성**을 조사하여 PO의 Go/No-Go 의사결정을 지원한다.

## 작업 흐름

```
조사설계가 → 4명 병렬 (시장/경쟁/고객/실현가능성) → 검증 → 전략가 종합
```

## 투입 시점

Stage 1. 사전조사

## 산출물

- 개별 분석 보고서 4개 (시장/경쟁/고객/실현가능성)
- 종합 사전조사 보고서 + Go/No-Go 판단 근거

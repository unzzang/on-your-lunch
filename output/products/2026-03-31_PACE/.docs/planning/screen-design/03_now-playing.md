# Now Playing 화면 설계

> 상태: 설계 중
> 디자인 파일: `design/pace-app-now-playing.pen`

---

## 화면 개요

학습 실행 중 전체화면. Spotify의 Now Playing 화면 벤치마크. 사용자가 집중해야 하는 화면이므로 **최소한의 정보만 표시** — 타이머와 과목명이 핵심.

## 화면 구조 (위→아래)

### 상단 바
- 좌측: ∨ 아래 화살표 (스와이프다운 → 미니 플레이어로 복귀)
- 중앙: 플레이리스트 제목 (작은 텍스트)
- 우측: ··· 더보기

### 커버 영역
- 대형 커버 이미지 (280x280, 중앙, radius 8px)
- 배경: 커버 이미지 기반 블러 그라데이션

### 세션 정보
- 현재 세션명: "수학 미적분" (Inter 24px bold, 흰색)
- 세션 타입: "학습 세션 3/10" (Inter 14px, #B3B3B3)

### 타이머 (핵심)
- 큰 타이머: "37:24" (Inter 56px bold, 흰색)
- 진행 바: 가로 프로그레스 바 (#1DB954 채움 / #282828 배경)
- 진행 바 좌측: 경과 시간 "12:36"
- 진행 바 우측: 전체 시간 "50:00"

### 컨트롤
- ⏮ 이전 세션 (skip-back, 32px)
- ⏸ 일시정지 / ▶ 재생 (원형 64px, 흰색 배경, 검정 아이콘)
- ⏭ 다음 세션 (skip-forward, 32px)

### 다음 세션 미리보기
- "다음: 휴식 10분" (Inter 14px, #6A6A6A)

### 진행도
- "세션 3/10 완료" (Inter 12px, #6A6A6A)
- 점 인디케이터: ●●●○○○○○○○ (완료=초록, 미완료=회색)

## 인터랙션

| 요소 | 동작 | 결과 |
|---|---|---|
| ∨ 아래 화살표 | 탭 | 미니 플레이어로 복귀 |
| 화면 전체 | 스와이프다운 | 미니 플레이어로 복귀 |
| 화면 전체 | 스와이프업 | 트랙 리스트 보기 |
| ⏸ 일시정지 | 탭 | 타이머 멈춤 |
| ▶ 재생 | 탭 | 타이머 재개 |
| ⏮ 이전 | 탭 | 이전 세션으로 |
| ⏭ 다음 | 탭 | 다음 세션으로 |
| 진행 바 | 드래그 | 세션 내 시간 탐색 (선택적) |

## 상태별 분기

| 상태 | 화면 차이 |
|---|---|
| 학습 세션 진행 중 | 기본 화면 |
| 휴식 세션 진행 중 | 세션명 "휴식", 커버 변경(차분한 이미지), 배경색 약간 다르게 |
| 일시정지 상태 | ⏸ → ▶ 변경, 타이머 멈춤, "일시정지 중" 표시 |
| 세션 전환 중 | 완료 메시지 + 5초 카운트다운 + [바로 시작] [스킵] |
| 15분 초과 | 팝업: "이어서 하기" / "다음 세션으로" / "오늘은 여기까지" |
| 마지막 세션 완료 | 완료 바텀시트 (축하 + 요약 + 복습 제안) |

## 디자인 원칙
- **최소 정보:** 타이머 + 과목명만 크게, 나머지는 작게
- **집중 유도:** 어두운 배경, 밝은 타이머, 시선이 중앙으로
- **Spotify Now Playing:** 커버 이미지 + 프로그레스 바 + 컨트롤 레이아웃 동일

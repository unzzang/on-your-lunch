# .infra/ 회사 운영용 코드 관리 규칙

## 역할

`.infra/`는 **회사 운영에 필요한 코드**를 관리하는 공간이다. 프로젝트별 코드(`src/`)와 구분한다.

```
.infra/ = 회사 운영용 코드 (내부 도구, 웹사이트, 공통 라이브러리)
src/    = 개별 프로젝트 코드 (앱, 서비스)
```

## 폴더 구조

```
.infra/
├── tools/       ← 내부 도구 (사내 대시보드, 자동화 스크립트 등)
├── website/     ← 회사 웹사이트 (소개, 채용 등)
└── shared/      ← 공통 라이브러리 (여러 프로젝트에서 재사용)
```

필요 시 폴더 추가 가능 (CI/CD, 배포 스크립트 등).

## 회사 vs 프로젝트 구분 기준

| 내용 | 저장 위치 |
|------|----------|
| 사내 자동화 스크립트 | `.infra/tools/` |
| 회사 소개 웹사이트 | `.infra/website/` |
| 여러 프로젝트에서 공유하는 라이브러리 | `.infra/shared/` |
| PACE 앱 코드 | 프로젝트 `src/` |
| on-your-lunch 앱 코드 | 프로젝트 `src/` |

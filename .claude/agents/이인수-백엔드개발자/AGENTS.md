# 이인수 — 백엔드 개발자

백엔드 아키텍트(이명환)의 설계를 기반으로 서버 사이드 코드를 구현하는 백엔드 개발자. API 엔드포인트, 인증 로직, 배치잡을 직접 구현한다.

## 성격

- 명세서와 ERD를 먼저 확인한 후 구현에 들어간다. 명세 없이 추측으로 구현하지 않는다.
- 테스트 코드를 함께 작성한다. 핵심 비즈니스 로직은 반드시 단위 테스트를 포함한다.
- 에러 핸들링을 꼼꼼히 처리한다. 예외 상황을 "나중에"로 미루지 않는다.

## 담당 범위

- **API 구현** — 엔드포인트별 컨트롤러, 서비스, 리포지토리 코드 작성
- **인증/인가 구현** — JWT, 소셜 로그인, 권한 가드 구현
- **배치잡 구현** — 스케줄링 작업, 데이터 처리 파이프라인
- **DB 마이그레이션** — 스키마 변경 마이그레이션 코드 작성 및 실행
- **단위/통합 테스트** — 비즈니스 로직, API 엔드포인트 테스트 코드 작성
- **API 문서화** — Swagger/OpenAPI 기반 API 문서 작성

# PACE 정보 구조도(IA) 및 메인 UX 설계

## 1. 앱 디자인 원칙: "친숙함이 곧 최고의 혁신"

- **100% Spotify UI 벤치마킹:** 수험생에게 새로운 앱 사용법을 학습시키는 것은 큰 장벽입니다. 사람들이 매일 숨 쉬듯 사용하는 '음악 스트리밍 앱'의 UI 문법을 그대로 차용하여 **학습 곡선(Learning Curve)을 0에 가깝게** 만듭니다.
- 공부(Study) = 음악 감상(Listening)
- 학습법/루틴 = 플레이리스트(Playlist)
- 과목/세부 태스크 = 개별 곡(Track)

## 2. 화면 구조 상세 (IA)

### 2.1. 상단 바 (Top Bar)

- **요소:** 마이페이지 진입 버튼 (좌측 상단, 아이디 첫 글자 프로필 아이콘)
- **기능:** 탭 시 좌측에서 드로어(Drawer) 메뉴 오픈.
- **드로어 메뉴 항목:**
  - 기본 스탯 (현재까지 공부한 총 시간, 스트릭 등)
  - 앱 설정 (다크모드, 알림 설정, 구독 관리 등)
  - 고급 사용자 설정 (과목별 비중 조절, 휴식 간격 세팅 등)

### 2.2. 중앙 영역 (Main Content Area)

음악 앱의 '추천 앨범', '오늘의 플레이리스트'와 완벽히 동일한 그리드/캐러셀 구조.

- **요소:** 공부 방식(루틴)을 담은 사각형 '카드(Card)' 리스트.
- **카드 디자인:**
  - 감각적이고 직관적인 앨범 커버 풍 이미지.
  - 제목 (예: "아침을 깨우는 하프 모의고사", "독하게 파고드는 수학 3시간")
  - 짧은 설명 (예: "시험 직전, 멘탈 관리를 위한 실전 모의고사 루틴")
- **인터랙션:** 카드를 터치하면 해당 공부 루틴의 '상세 페이지(곡 목록)'로 이동하여, 내부의 세부 과목(곡)과 소요 시간 확인 가능.

### 2.3. 하단 내비게이션 바 (Bottom Navigation Bar)

앱의 주요 4가지 탭 이동.

1.  **홈 (Home):** (현재 탭) 다양한 AI 추천 공부 설정 카드 큐레이션 제공.
2.  **검색 (Search):** (돋보기 아이콘) 카테고리별 탐색 및 특정 키워드로 공부 루틴(카드) 검색.
3.  **내 라이브러리 (Library):** (책 모양 아이콘) 내가 '좋아요(하트)'를 눌러 보관한 카드, 또는 내가 직접 생성한 루틴(카드) 목록 조회.
4.  **만들기 (Create):** (플러스 아이콘) 나만의 공부 루틴 카드를 직접 커스텀하여 생성 (제목, 커버, 세부 트랙 조합). 고급 사용자를 위한 영역.

### 2.4. 플로팅 플레이어 (Floating Player) [🔥 핵심 UX]

음악 앱처럼 하단 내비게이션 바 바로 위에 항상 떠 있는 플로팅 영역.

- **기능:** 현재 활성화된(가장 최근에 재생한) 공부 카드를 언제든 즉시 통제할 수 있는 **'상시 대기 조종석'**.
- **노출 요소:**
  - 햔재 플레이 중인(또는 대기 중인) 카드 썸네일 이미지
  - 카드 타이틀 및 현재 진행 중인 세부 과목(Track) 텍스트
  - 조작 버튼: `일시정지(Pause)` / `재생(Play)` 버튼
- **인터랙션:** 이 영역을 터치(Swipe-up)하면 화면 전체를 덮는 매끈한 '현재 플레이 화면(Now Playing)'으로 확장되며, 거대한 타이머(진행 바)와 다음 세션 목록 노출.

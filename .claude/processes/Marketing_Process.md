# 마케팅 프로세스

> 적용 범위: GTM 전략 수립부터 채널 운영까지
> 책임자: PO (의사결정), 한지민 (전략), 이수빈 (채널 운영)
> 선행 조건: Stage 2 Gate 1 완료 (서비스 정의 확정)

---

## 1. 역할

서비스의 시장 진입 전략을 수립하고, 채널을 운영하며, 데이터 기반으로 성과를 개선한다.

## 2. 프로세스

```
Phase 1 → Phase 2 → Phase 3 → Phase 4 (반복)
```

### Phase 1. GTM 전략 수립 (한지민)

- 타겟 재검증 (`.docs/003_research/` 참조)
- 채널 선정 + 예산 배분
- 핵심 메시지 설계
- 전략 문서 → PO 승인 → DR 기록

### Phase 2. 콘텐츠 연계

- 콘텐츠팀(강서연, 류현우)에 마케팅 방향 전달
- 디자인팀 박소율에 광고 크리에이티브 요청
- 콘텐츠 일정과 마케팅 일정 동기화

### Phase 3. 채널 운영 (이수빈)

- SNS 콘텐츠 일정 관리 + 발행
- 커뮤니티 대응
- 성과 측정 → 데이터팀(톰크루즈)과 협업

### Phase 4. 성과 분석 + 개선

- 채널별 KPI 추적 (CAC, 전환율, 리텐션 등)
- 데이터 기반 채널 최적화
- 한지민 → 전략 업데이트 → PO 보고

## 3. 완료 기준

- [ ] GTM 전략 문서 확정 + DR 기록
- [ ] 채널 운영 개시
- [ ] 성과 측정 체계 구축
- [ ] 최소 1회 성과 분석 → 개선 사이클 완료

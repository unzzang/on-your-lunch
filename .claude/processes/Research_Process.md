# 사전조사 프로세스 (Stage 1)

> 적용 범위: 서비스 기획 전 시장/경쟁/고객/실현가능성 조사
> 책임자: PO (지시), 리서치팀 6명 (수행)
> 선행 조건: Stage 0 완료

---

## 1. 역할

서비스 아이디어의 시장성, 경쟁 환경, 고객 니즈, 실현가능성을 조사하여 Go/No-Go 판단 근거를 확보한다.

## 2. 프로세스

```
Phase 0 → Phase 1 → Phase 2 (4명 병렬) → Phase 3 → Phase 4
```

### Phase 0. PO 인터뷰

PO가 아래 4가지에 답한다:

1. 어떤 문제를 해결하려는가?
2. 누구를 위한 서비스인가?
3. 왜 지금 해야 하는가?
4. 경쟁사 대비 어떻게 다른가?

### Phase 1. 조사 설계 (박선우)

- KRQ(Key Research Question) 수립
- 가설 설정
- 4개 영역별 조사 범위 확정

### Phase 2. 병렬 리서치

| 영역 | 담당 | 산출물 위치 |
|------|------|------------|
| 시장 분석 | 정하은 | `.docs/003_research/` |
| 경쟁 분석 | 한도윤 | `.docs/003_research/` |
| 고객 리서치 | 윤서진 | `.docs/003_research/` |
| 실현가능성 | 최재현 | `.docs/003_research/` |

### Phase 3. 검증 (송현아)

- 4개 리서치 보고서의 정합성, 논리 누락, 근거 부족 검증
- 문제 발견 시 해당 담당자에게 보완 요청

### Phase 4. 전략 종합 (오민석)

- 4개 보고서 + 검증 결과를 종합
- Go/No-Go 권고 → PO 최종 결정

**운영 방식:** PO가 리서치팀에 직접 지시하거나, 위더에게 전체 조율을 맡길 수 있다.

## 3. 완료 기준

- [ ] 4개 영역 리서치 보고서 완료
- [ ] 검증 통과 (송현아)
- [ ] 전략 종합 보고서 완료 (오민석)
- [ ] Go/No-Go 결정 → DR 기록 (`.docs/001_decision-making/`)

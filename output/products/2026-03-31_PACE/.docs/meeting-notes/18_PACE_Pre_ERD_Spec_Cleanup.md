## 발견 및 수정된 명세서 불일치 (5건)

### 1. exam_type 코드값 불일치 — `04_search.md` 수정

**문제:** 온보딩(ONB-002)에서 사용하는 `exam_type` 값(`suneung`, `civil_service`, `certification` 등)과 Search 화면의 카테고리 ID(`cat_suneung`, `cat_civil`, `cat_cert` 등)가 달라 매핑 관계가 불명확했음.

**조치:** SRC-001에 카테고리 ID → exam_type 매핑 테이블 추가.

| 카테고리 ID | exam_type 값 |
|---|---|
| cat_suneung | suneung |
| cat_civil | civil_service |
| cat_cert | certification |
| cat_lang | language |
| cat_transfer | transfer |

> `other` 타입은 카테고리 없음. 검색으로만 탐색.

---

### 2. NP-009 집중률 공식 불명확 — `07_now-playing.md` 수정

**문제:** 완료 바텀시트의 "집중률: 78%"가 일간 집중률인지 세션 집중률인지 명시 없음. FC-002에서 확정한 두 공식 중 어느 것인지 불명확했음.

**조치:** NP-009에 "(세션 집중률: 순공 시간 / 배정 총 시간 × 100, FC-002 참조)" 명시.

---

### 3. 연속 학습일 "세션" 정의 불명확 — `08_mypage.md` 수정

**문제:** MP-003에 "하루 1개 세션 이상 완료"가 개별 학습 트랙 완료인지, 플레이리스트 전체 완료인지 불명확했음.

**조치:** "학습 트랙(세션) 1개 이상 완료 = 당일 학습 인정. 플레이리스트 전체 완료 여부와 무관. 타이머 0 도달 시 완료, 중도 종료는 미완료"로 명확화.

---

### 4. Library "수정본" 배지 UI 위치 미정의 — `05_library.md` 수정

**문제:** LIB-002에 "수정본 배지 표시"라고만 되어 있고 위치가 명시되지 않아 개발/디자인 해석 불일치 가능성.

**조치:** "커버 이미지 우측 하단에 오버레이 (accent 색 텍스트 + 어두운 반투명 배경)"로 위치 명시.

---

### 5. HOME-002 → Fact-Check 라우팅 불명확 — `03_home.md` 수정

**문제:** HOME-002에 "MyPage > 학습 스탯 상세 (Fact-Check)"라고 표기되어 MyPage 화면을 경유하는 것처럼 오해될 수 있었음. FC-001·MP-001은 직접 이동으로 기술되어 있어 불일치.

**조치:** "Fact-Check 리포트 화면으로 직접 이동 (MyPage 화면 경유 없음)"으로 수정.

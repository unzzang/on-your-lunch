## 범위 구분

- **O** = MVP 필수 (없으면 서비스가 성립하지 않음)

## 핵심 사용자 플로우 (MVP)

```
1. 앱 열기 → 소셜 로그인 → 회사 위치 + 취향 설정 (최초 1회)
2. 11:30 점심 시간 → 앱 진입
3. AI가 날씨/계절/시간 + 가격대 기반으로 메뉴 4개 추천
4-A. [그룹] "이걸로 투표하기" 탭 → 투표방 자동 생성
5. 카카오톡으로 동료에게 초대 링크 공유
6. 동료들이 링크 열고 투표 (앱 미설치자도 웹으로 참여 가능)
7. 5분 타이머 후 자동 마감 → 1등 메뉴 확정
8. "배달 주문하기" → 배민/쿠팡이츠로 딥링크 이동
9. 오늘의 점심 자동 기록
4-B. [혼자] AI 추천 메뉴 중 1개 선택 → 배달앱 딥링크 이동 + 점심 기록 자동 생성
```

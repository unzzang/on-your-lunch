# 기능 명세서: Common (공통 컴포넌트)

> 최종 수정: 2026-03-17
> 상태: 초안

## COM-001. 하단 네비게이션 바

### 탭 구성

| 탭 | 아이콘 | 화면 |
|---|---|---|
| Home | 홈 아이콘 | Home 화면 |
| Search | 돋보기 아이콘 | Search 화면 |
| Library | 보관함 아이콘 | Library 화면 |

### 상태
- 활성 탭: accent 색상 (#4A90D9) + 탭 이름 텍스트
- 비활성 탭: text-muted 색상 (#6A6A6A) + 아이콘만

### 전환 동작
- 탭 탭 → 해당 화면으로 즉시 전환 (탭 간 애니메이션 없음)
- 같은 탭 재탭 → 해당 화면 최상단으로 스크롤

### 노출 조건
- 항상 노출 (Home, Search, Library)
- 미노출: Now Playing 전체화면, 온보딩, Auth 화면

### 미니 플레이어와의 관계
- 재생 중: 미니 플레이어가 네비 바 위에 위치
- 미재생: 네비 바만 노출

## COM-002. 미니 플레이어

### 노출 조건
- 플레이리스트 재생 중 (진행 중인 세션이 있을 때)
- 일시정지 상태에서도 노출
- Now Playing 전체화면에서는 숨김

### 표시 정보
- 좌측: 커버 이미지 미니 (40x40)
- 중앙: 현재 세션명 + 남은 시간 (1줄)
- 우측: ⏸/▶ 토글 버튼

### 인터랙션

| 동작 | 결과 |
|---|---|
| 미니 플레이어 탭 | Now Playing 전체화면 진입 |
| 스와이프업 | Now Playing 전체화면 진입 |
| ⏸/▶ 버튼 | 타이머 일시정지/재개 |

### 위치
- 하단 네비 바 바로 위에 고정
- 높이: 56px
- 배경: bg-elevated (#1A1A1A)

### 세션명 말줄임
- "수학 미적분 — 37:24 남음" (1줄 초과 시 세션명 말줄임)

## COM-003. 토스트 메시지

### 표시 방식
- 화면 하단에서 올라오는 스낵바
- 자동 사라짐: 3초 (기본)
- Undo 포함 토스트: 5초

### 종류

| 유형 | 예시 | 액션 |
|---|---|---|
| 성공 | "보관함에 저장했어요" | 없음 |
| 실패 | "네트워크 연결을 확인해주세요" | 없음 |
| Undo | "세션이 삭제되었어요" | [되돌리기] 버튼 |

### 겹침 정책
- 동시 2개 이상: 새 토스트가 기존 토스트를 교체 (큐 방식 아님)

## COM-004. 확인 팝업 (다이얼로그)

### 공통 구조
- 중앙 모달
- 배경 딤 처리 (#000000 50%)
- 제목 (선택사항) + 메시지 + 버튼 1~2개

### 버튼 배치
- 2버튼: [취소(왼)] [확인(오른)]
- 확인 버튼: 일반 = accent 색상, 위험 = error 색상
- 취소 버튼: 항상 text-secondary

### 팝업 외 영역 탭
- 기본: 팝업 닫힘 (취소와 동일)
- 중요 팝업 (탈퇴, 로그아웃): 외부 탭 무시

## COM-005. 바텀시트

### 공통 구조
- 하단에서 올라오는 시트
- 드래그 핸들 (상단 바)
- 스와이프다운 → 닫기
- 배경 딤 처리

### 높이
- 콘텐츠에 따라 자동 조절
- 최대 높이: 화면의 80%
- 내용 초과 시 시트 내 스크롤

---

## COM-006. 로딩 상태

### 화면 최초 로드
- 스켈레톤 UI (카드/리스트 형태의 회색 플레이스홀더)
- 이미지: bg-surface (#242424) 플레이스홀더 → 이미지 로드 완료 시 페이드인

### 추가 데이터 로드 (무한 스크롤)
- 리스트 하단 스피너 (원형 로딩 인디케이터)

### 에러 상태 (데이터 로드 실패)
- "데이터를 불러오지 못했어요" + [다시 시도] 버튼

---

## COM-007. 오프라인 상태

### 감지
- 네트워크 연결 끊김 시 상단 배너: "오프라인 상태예요" (warning 색상)
- 연결 복구 시 배너 자동 사라짐

### 오프라인에서 가능한 것
- Now Playing 타이머 동작 (로컬)
- 이전에 로드된 화면 캐시 표시

### 오프라인에서 불가능한 것
- 새 데이터 로드 (API 호출)
- 좋아요/저장 (로컬 큐잉 → 온라인 복귀 시 동기화)

## COM-008. 딥링크

### 지원 링크 (MVP)

| 링크 패턴 | 이동 화면 |
|---|---|
| `pace://playlist/{id}` | 플레이리스트 상세 |
| `pace://home` | Home |
| `pace://reset-password?token={token}` | 비밀번호 재설정 |

### 비로그인 상태에서 딥링크
- 로그인 화면 → 로그인 완료 → 원래 딥링크 목적지로 이동

## COM-009. 앱 시작 플로우

```
앱 실행
├─ Refresh Token 유효? ─── Yes ──→ 자동 로그인
│                                    ├─ 온보딩 미완료 → 온보딩
│                                    └─ 온보딩 완료 → Home
└─ No (만료/없음) ─────────────────→ Auth 로그인 화면
```

### 스플래시 화면
- PACE 로고 + 배경 (#121212)
- 표시 시간: 토큰 검증 완료까지 (최소 1초, 최대 3초)
- 3초 초과 시 로그인 화면 이동 (타임아웃)
